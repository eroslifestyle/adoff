### Metodo Sourcing Tier C — Raggiungi 50

**Dopo i 25 concreti, replica questo workflow per altri ~25:**

**Fase 1: Search Query Database** (~5-10 canali per query)
```
YouTube search queries:
- "Linux ad blocker extension"
- "privacy extension review" 
- "ublock origin alternatives"
- "best chrome privacy extensions"
- "Linux tutorial 2026" (filter 10K-100K subs)
- "Arch Linux setup guide"
- "privacy YouTube channels"
- "FOSS creator"
```

**Fase 2: Hashtag + Community Mining** (~10 per hashtag)
```
YouTube/TikTok hashtags:
- #PrivacyTech #OpenSource #Linux #FOSS #CyberSecurity #PrivacyMatters
- #FossCreator #LinuxTutorial #ArchLinux
```

**Fase 3: Related Channels (YouTube "People Also Watch")** (~5-10 per channel)
- Entra in ogni Tier B → scheda "Channels you might like" → estrai nomi/subs

**Fase 4: Community Post Mining** (~5-10 per creator)
- Scorri Tier B community posts → guarda chi commenta frequentemente → estrai account creator

**Fase 5: Discourse/Forum Scrape** (~10-15)
```
Forum:
- https://discuss.privacyguides.net
- https://discuss.techlore.tech
- https://community.learnlinux.tv
- Arch Linux Forums
- Linux subreddits (r/ArchLinux, r/OpenBSD, r/privacy)
```
Cerca usernames attivi che linkano blog/YouTube → estrai channel.

**Fase 6: GitHub Awesome-Lists** (~10)
- Cerca "awesome-privacy" + "awesome-linux" → guarda contributors → molti hanno YouTube/blog

## Tracking & Metriche Outreach

### Per ogni tier:
- **Contatti inviati** (data, channel, message version)
- **Risposte ricevute** (date, sentiment, next step)
- **Content prodotto** (video published date, link, mentions)
- **Referral codes redeemed** (codice unico → conversioni → LTV tracking)
- **Click-through / share metrics** (YouTube community post engagement, blog link clicks via ref code)

### Sistema tracking integrato:
1. **Referral code per creator**: `CREATORNAME-Q2-2026` (es. `NAOMIBROCKWELL-Q2-2026`)
2. **Backend**: Codice → storage `adoffReferralHistory` (vedi CLAUDE.md § Storage Keys) → API tracking conversioni giornaliere
3. **Dashboard**: Spreadsheet Google Sheet con colonne: Creator | Sent Date | Code | Clicks | Conversions | LTV | Status
4. **Cadenza**: Aggiornamento settimanale via script cron (daily sync referral data → sheet)

### ROI Baseline (per progettare fase 2):
- **Tier A**: 0-2 conversioni dirette attese; goal = earned brand mentions, credibilità
- **Tier B**: 3-8 referral conversions per creator (gifting access = abbassa barrier)
- **Tier C**: 1-2 conversioni per micro-creator (50 creator target = 50-100 conversioni totali)
- **Tier D**: +10-20 referrals per sito major (SEO traffic alto)

**Totale ipotizzato Fase 1**: 80-150 conversioni referral (mix Tier A earned + B/C/D affiliate), LTV medio €60-120 per conversione (stima).
