🛠 1. Apa itu AdOff

AdOff adalah ekstensi untuk Google Chrome yang memblokir iklan di semua situs web. Bekerja pada empat lapisan — jaringan, halaman, netralisasi iklan video (hanya Pro/Trial), dan stealth (hanya Pro/Trial) — untuk pengalaman browsing yang bersih dan cepat.

  • Memblokir iklan di situs mana pun; netralisasi iklan video di platform streaming (Pro/Trial)
  • Mode Stealth (Pro/Trial): membuat AdOff tak terlihat oleh sistem anti-adblock
  • Super ringan — 20 kali lebih kecil dari pesaing
  • Tidak mengumpulkan data pribadi: semua berjalan secara lokal
  • 6 bahasa: Italia, Inggris, Jerman, Prancis, Spanyol, Portugis
  • Jeda per situs dengan 4 durasi: kunjungan ini, 1 jam, 1 hari, permanen
  • Sistem referral untuk mendapatkan hari Pro gratis

📥 2. Instalasi

Dari Chrome Web Store

  1. Kunjungi halaman AdOff di Chrome Web Store
  2. Klik "Tambahkan ke Chrome"
  3. Konfirmasi dengan "Tambahkan ekstensi"
  4. AdOff langsung aktif

Menyematkan ikon di toolbar

  1. Klik ikon puzzle (Ekstensi) di kanan atas browser
  2. Temukan AdOff dalam daftar
  3. Klik pin di sebelah AdOff
  4. Ikon muncul di toolbar untuk akses cepat

🎉 3. Pertama kali digunakan

Saat pertama kali menginstal AdOff:

  • Sebuah halaman selamat datang terbuka dengan instruksi untuk menyematkan ikon
  • Trial Pro 30 hari diaktifkan secara otomatis
  • Sebuah kode referral unik dibuat (format: ADO-XXXXX)
  • AdOff mulai memblokir iklan secara langsung
  • Jika Anda menginstal sebelum 1 Juli 2026, Anda mendapatkan badge Founding Member permanen

🖼 4. Popup

Klik ikon AdOff di toolbar untuk membuka popup.

Header

Logo, nama "AdOff", badge lisensi (FREE / TRIAL Xhari / PRO) dan badge FOUNDER (jika memenuhi syarat).

Toggle global

Tombol ON/OFF untuk mengaktifkan atau menonaktifkan AdOff di semua situs. Badge ungu = aktif, merah = nonaktif.

Penghitung

  • Iklan diblokir: elemen iklan yang dihapus dari halaman
  • Permintaan diblokir: permintaan HTTP ke server iklan yang dicegat

Angka disingkat: 1.500 menjadi "1.5K", 2.300.000 menjadi "2.3M".

Situs saat ini

Domain situs yang sedang Anda kunjungi, dengan tombol "Jeda di sini".

Banner lisensi

Hanya terlihat untuk pengguna Free ("Coba Pro 30 hari gratis") dan Trial ("Trial aktif — X hari tersisa"). Tersembunyi untuk pengguna Pro.

Prompt ulasan

Muncul setelah 100+ iklan diblokir dan 5+ hari sejak instalasi. Maksimal 3 kemunculan, dengan jeda 14 hari di antara masing-masing. Tiga opsi: "Tinggalkan ulasan", "Nanti saja", "Jangan pernah lagi".

Changelog

Setelah pembaruan, menampilkan hal baru. Tutup dengan X.

Tautan di bagian bawah

Bantuan (membuka FAQ di opsi) dan Opsi (membuka halaman lengkap).

⏸ 5. Jeda per situs

Nonaktifkan AdOff di satu situs tanpa mematikannya di mana-mana.

Cara menjeda

  1. Kunjungi situs
  2. Klik ikon AdOff di toolbar
  3. Klik "Jeda di sini"
  4. Pilih durasi
OpsiYang terjadi
Hanya kunjungan iniDiaktifkan kembali saat Anda menutup tab
1 jamDiaktifkan kembali setelah 60 menit
1 hariDiaktifkan kembali setelah 24 jam
Selalu (whitelist)Tetap dinonaktifkan secara permanen

Cara mengaktifkan kembali

Kunjungi situs yang dijeda, klik ikon AdOff, Anda akan melihat "Dijeda — [tipe]", klik "Aktifkan kembali".

Catatan: jeda yang sudah kedaluwarsa dihapus secara otomatis. Pemblokiran efektif juga berlaku untuk subdomain: jika Anda menjeda example.com, blog.example.com juga dikecualikan.

🛡 7. Situs dikecualikan (Whitelist)

Menambahkan situs

  1. Pergi ke Opsi > Situs dikecualikan
  2. Ketik domain (mis. example.com)
  3. Klik "Tambahkan" atau tekan Enter

Ekspor dan impor

Ekspor: unduh whitelist sebagai file JSON. Impor: muat file JSON. Entri divalidasi (domain maks 255 karakter, tipe yang diizinkan). Entri tidak valid diabaikan.

⚠ 13. Laporkan masalah

Jika suatu situs tidak berfungsi dengan AdOff, laporkan di Opsi > Bantuan > Laporkan masalah.

  1. Masukkan URL situs (wajib)
  2. Pilih tipe: Situs tidak berfungsi / Iklan masih terlihat / Anti-adblock / Lainnya
  3. Tambahkan deskripsi dan email (opsional)
  4. Selesaikan captcha matematika
  5. Klik "Kirim laporan"

Formulir memiliki perlindungan anti-bot: honeypot, timing, captcha, rate limit (maks 3/jam), anti-spam, deduplikasi.

💡 14. Saran

Di Opsi > Saran Anda dapat mengusulkan fitur, melaporkan bug, atau perbaikan. Isi judul (wajib), deskripsi, dan email (opsional). Saran Anda muncul dalam daftar di bawah formulir.

⚙ 15. Cara kerja AdOff (teknis)

Lapisan 1 — Pemblokiran jaringan

AdOff menggunakan API declarativeNetRequest Chrome untuk memblokir permintaan HTTP ke server iklan sebelum browser mengunduhnya. Lebih dari 120 aturan mencakup jaringan iklan utama dan banyak ad network lainnya.

Lapisan 2 — Filter kosmetik

Untuk iklan yang disajikan dari domain yang sama dengan situs, AdOff menggunakan CSS dan JavaScript untuk menyembunyikannya dari halaman: selektor yang presisi, MutationObserver untuk konten dinamis, kolaps kontainer kosong.

Lapisan 3 — Netralisasi iklan video pada platform streaming (hanya Pro/Trial) PRO

Pada platform streaming, sebagian besar pemutar video menggunakan SDK iklan standar untuk mengelola iklan video (pre-roll, mid-roll, post-roll). AdOff Pro menggantikan SDK ini dengan versi netral yang langsung memberi tahu pemutar "tidak ada iklan" — video berjalan tanpa gangguan. Fitur ini bekerja pada platform streaming yang menggunakan SDK standar. Fitur eksklusif Pro.

Lapisan 4 — Mode Stealth (hanya Pro/Trial)

Mencegat pemeriksaan anti-adblock dan membuat situs percaya bahwa segalanya terlihat:

  • Spoofing CSS: properti elemen umpan dipalsukan
  • Intercept jaringan: skrip deteksi menerima respons kosong
  • Variabel yang disimulasikan: SDK iklan diisi sebelumnya
  • Netralisasi skrip: skrip anti-adblock diblokir sebelum dieksekusi
  • Anti scroll-lock: mencegah tembok memblokir scroll

Mode Stealth dinonaktifkan secara otomatis di platform utama (video, sosial, e-commerce) dan situs besar lainnya di mana hal itu akan mengganggu.

🛠 16. Pemecahan masalah

Suatu situs tidak berfungsi

  1. Klik ikon AdOff dan gunakan "Jeda di sini"
  2. Muat ulang halaman
  3. Jika berfungsi saat dijeda: laporkan situs kepada kami
  4. Jika tidak berfungsi bahkan saat dijeda: masalahnya bukan dari AdOff

Saya masih melihat iklan

  1. Verifikasi bahwa AdOff aktif (badge ungu)
  2. Verifikasi bahwa situs tidak ada di whitelist
  3. Muat ulang halaman
  4. Jika masih ada: laporkan situs kepada kami

"Nonaktifkan ad blocker Anda"

  1. Muat ulang halaman
  2. Pastikan Anda memiliki Pro atau Trial (Mode Stealth diperlukan)
  3. Jika berlanjut: gunakan "Jeda di sini" dan laporkan kepada kami

AdOff tidak aktif

Pergi ke chrome://extensions/, verifikasi bahwa ekstensi diaktifkan. Jika ada kesalahan, coba hapus dan instal ulang.

🗑 19. Copot instalasi

  1. Pergi ke chrome://extensions/
  2. Temukan AdOff dan klik "Hapus"
  3. Konfirmasi

Pengaturan lokal dihapus. Lisensi Pro tetap valid di server. Kode referral hilang.

Sebelum mencopot instalasi, ekspor pengaturan dari Opsi > Lanjutan > Ekspor. Setelah menginstal ulang, impor file untuk memulihkan segalanya (lisensi harus diaktifkan kembali dengan kunci).
