Cara Berkontribusi

AdOff dibentuk oleh komunitasnya. Setiap kontribusi, besar atau kecil, membuat perbedaan.

🐛
Laporkan Bug
Menemukan iklan yang lolos? Situs yang rusak? Buka laporan bug di GitHub atau Discord dan kami akan menyelidikinya. Sertakan URL situs dan langkah-langkah untuk mereproduksi.
Buka laporan →
💡
Usulkan Fitur
Punya ide yang akan membuat AdOff lebih baik? Bagikan di diskusi GitHub atau Discord kami. Ide terbaik masuk ke rilis berikutnya.
Bagikan ide Anda →
⭐
Tulis Ulasan
Ulasan di Chrome Web Store membantu lebih banyak pengguna menemukan AdOff. Hanya butuh 30 detik dan membuat perbedaan besar untuk proyek independen seperti kami.
Tinggalkan ulasan →

Gunakan tautan referral pribadi Anda dari Opsi AdOff untuk mengundang teman. Saat mereka beralih ke Pro, keduanya mendapat hari Pro gratis — tidak ada batas berapa banyak yang bisa Anda dapatkan.
