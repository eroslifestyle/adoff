Kebijakan Privasi

Terakhir diperbarui: 21 April 2026  ·  Versi 2.0

Singkatnya: AdOff tidak mengumpulkan data browsing pribadi. Tidak melacak Anda. Tidak menjual data kepada siapa pun. Satu-satunya komunikasi dengan server kami adalah validasi lisensi Pro — dan bahkan di situ, kami hanya berbagi yang paling minimal. Versi GRATIS bekerja tanpa pendaftaran atau akun — tidak ada data yang dikumpulkan dari pengguna gratis selain yang tersimpan secara lokal di perangkat.

1. Siapa kami

AdOff adalah ekstensi browser yang memblokir iklan dan tracker. Pengontrol data adalah tim AdOff, yang dapat dihubungi melalui halaman dukungan.

2. Data yang tidak kami kumpulkan

AdOff tidak mengumpulkan, tidak mengirimkan, dan tidak menyimpan:

  • Riwayat penelusuran atau URL situs yang dikunjungi
  • Konten halaman atau data yang dimasukkan ke formulir
  • Cookie pelacakan atau pembuatan profil dari pihak ketiga
  • Data perilaku atau pembuatan profil
  • Data apa pun yang dapat digunakan untuk membuat profil Anda tanpa persetujuan Anda

Versi GRATIS bekerja tanpa pendaftaran atau akun. Tidak ada data yang dikumpulkan dari pengguna gratis selain yang tersimpan secara lokal di perangkat.

3. Data akun (khusus pengguna Pro)

Jika Anda membuat akun atau membeli paket Pro, kami mengumpulkan data berikut:

  • Alamat email — dikumpulkan saat pembelian (melalui Stripe) dan pendaftaran akun. Digunakan untuk: pengiriman lisensi, login akun, komunikasi layanan. Dasar hukum: pelaksanaan kontrak (Art. 6(1)(b) GDPR).
  • Kata sandi — disimpan sebagai hash tidak dapat dikembalikan (SHA-256 + salt). Tidak pernah dalam teks biasa. Tidak pernah dibagikan kepada pihak ketiga.
  • Google OAuth — jika Anda menggunakan "Masuk dengan Google", kami menerima email profil Google dan ID Anda. Kami tidak mengakses data lain dari akun Google Anda.
  • UUID perangkat — pengenal acak yang dibuat saat instalasi ekstensi. Digunakan untuk menerapkan batas perangkat per lisensi. Tidak terhubung dengan identitas Anda. Dasar hukum: kepentingan sah (Art. 6(1)(f) GDPR).

4. Data yang disimpan secara lokal

AdOff menggunakan chrome.storage.local dan chrome.storage.sync untuk menyimpan secara lokal, di perangkat Anda:

  • Kunci lisensi Pro (jika dibeli)
  • Preferensi ekstensi (mis. whitelist situs, pengaturan UI)
  • Status aktivasi (aktif/nonaktif per situs)

Data ini tidak pernah dikirim ke server pihak ketiga dan tetap ada di browser Anda. chrome.storage.sync dapat menyinkronkan data antar perangkat melalui akun Google Anda, sesuai kebijakan Google.

5. Validasi lisensi

Jika Anda telah membeli paket Pro, ekstensi menghubungi server kami untuk memverifikasi validitas lisensi. Dalam komunikasi ini yang dikirimkan adalah:

  • Kunci lisensi
  • UUID perangkat
  • Pengenal browser/OS (untuk dasbor manajemen perangkat)
  • Versi ekstensi

Alamat IP Anda terlihat oleh server kami (Cloudflare) tetapi tidak disimpan secara permanen — hanya geolokasi tingkat negara/kota yang diturunkan untuk analitik agregat. Log akses server dihapus dalam 30 hari.

6. Pihak ketiga

AdOff tidak berbagi, tidak menjual, dan tidak menyerahkan data kepada pihak ketiga untuk tujuan pemasaran atau pembuatan profil. Kami tidak menggunakan:

  • Layanan analitik perilaku
  • Layanan periklanan
  • SDK pelacakan pihak ketiga

Layanan pihak ketiga yang kami gunakan untuk memberikan layanan:

  • Stripe — pemrosesan pembayaran. Data pembayaran dikelola oleh Stripe sesuai Kebijakan Privasi mereka sendiri.
  • Google Sign-In — jika Anda menggunakan "Masuk dengan Google", autentikasi Anda dikelola oleh Google. Lihat kebijakan privasi Google.
  • Cloudflare — hosting dan pengiriman konten. Cloudflare dapat memproses alamat IP sesuai kebijakan privasi mereka.

7. Izin ekstensi

AdOff memerlukan izin browser berikut, semuanya diperlukan untuk berfungsi:

  • declarativeNetRequest — untuk memblokir permintaan jaringan (iklan, tracker)
  • storage — untuk menyimpan preferensi dan lisensi secara lokal
  • tabs — untuk menerapkan pengaturan ke tab aktif
  • scripting — untuk menyuntikkan CSS penyembunyian di halaman web
  • webNavigation — untuk menerapkan kembali filter saat Anda menavigasi antar halaman di tab yang sama

Tidak ada izin ini yang digunakan untuk mengumpulkan atau mengirimkan data browsing.

8. Hak Anda (GDPR)

Berdasarkan Regulasi Perlindungan Data Umum Uni Eropa, Anda memiliki hak untuk:

  • Akses — meminta salinan data apa pun yang kami miliki tentang Anda
  • Rektifikasi — meminta koreksi data yang tidak akurat
  • Penghapusan — meminta penghapusan data Anda ("hak untuk dilupakan")
  • Portabilitas data — menerima data Anda dalam format portabel
  • Pembatasan — meminta pembatasan pemrosesan
  • Keberatan — menolak pemrosesan data Anda
  • Keluhan — mengajukan keluhan kepada otoritas pengawas di Uni Eropa
  • Penghapusan akun — menghapus akun dan semua data terkait Anda
  • Ekspor data — mengekspor data akun Anda
  • Manajemen perangkat — mengelola perangkat berlisensi Anda dari adoff.app/account

Untuk menggunakan hak Anda, hubungi kami melalui halaman dukungan.

9. Retensi data

  • Preferensi lokal: disimpan di perangkat Anda hingga Anda mencopot instalasi ekstensi atau menghapus data browser.
  • Data akun (email, kata sandi yang di-hash): disimpan selama akun aktif. Dihapus atas permintaan.
  • UUID perangkat: dihapus saat Anda menghapus perangkat dari dasbor akun Anda.
  • Geolokasi yang berasal dari IP: dikumpulkan hanya di tingkat negara. Tidak ada alamat IP individu yang disimpan secara permanen.
  • Kata sandi yang di-hash: dihapus saat akun dihapus.
  • Log validasi lisensi: dihapus secara otomatis dalam 30 hari.
  • Data pembayaran: dikelola oleh prosesor pembayaran kami sesuai kebijakan retensi mereka.

10. Dasar hukum pemrosesan

Dasar hukum untuk memproses data akun dan lisensi adalah pelaksanaan kontrak (Art. 6(1)(b) GDPR) — hal ini diperlukan untuk memverifikasi lisensi Pro Anda dan memberikan layanan yang Anda beli. Pemrosesan UUID perangkat didasarkan pada kepentingan sah (Art. 6(1)(f) GDPR) — diperlukan untuk menerapkan batas perangkat per lisensi.

11. Anak di bawah umur

AdOff tidak ditujukan untuk orang di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan data dari anak di bawah umur.

12. Perubahan pada kebijakan ini

Kami mungkin memperbarui Kebijakan Privasi ini di masa mendatang. Perubahan signifikan akan dikomunikasikan melalui ekstensi atau situs web. Dengan terus menggunakan AdOff setelah perubahan, Anda menyetujui versi kebijakan yang baru.

14. Data Diagnostik Sukarela

AdOff menyertakan fitur pelaporan kebocoran iklan opt-in: ketika ekstensi mendeteksi bahwa iklan terlihat di halaman dan kamu memilih untuk melaporkannya, paket diagnostik dikirim ke server kami untuk dianalisis. Fitur ini sepenuhnya opsional dan dinonaktifkan secara default. Tidak ada data diagnostik yang pernah dikirim tanpa persetujuan eksplisit kamu. Saat kamu mengirimkan laporan, informasi berikut ditransmisikan:

  • URL halaman tempat iklan terdeteksi
  • Versi ekstensi AdOff yang terpasang
  • Data teknis tentang elemen iklan yang terlihat (selector CSS, posisi)
  • Daftar aturan filter aktif (bukan riwayat penjelajahan kamu)
  • Informasi browser dan sistem operasi dasar
Data ini digunakan hanya untuk meningkatkan aturan pemblokiran iklan. Tidak ditautkan ke identitasmu, dijual, atau dibagikan ke pihak ketiga mana pun. Laporan diagnostik disimpan maksimal 90 hari lalu dihapus.
