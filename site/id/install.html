1
Ekstrak file ZIP
Cari adoff-chrome.zip di Download.

Windows: Klik kanan → Ekstrak Semua → Ekstrak
Mac: Klik ganda pada ZIP. Folder akan dibuat secara otomatis.
Linux: Klik kanan → Ekstrak di sini atau unzip adoff-chrome.zip
⚠ Penting: Jangan hapus folder setelah pemasangan. Browser harus membacanya dari disk. Letakkan di tempat yang aman (mis. Dokumen).
2
Buka halaman Ekstensi Chrome
Buka Chrome dan ketik chrome://extensions di bilah, lalu tekan Enter.

Halaman dengan semua ekstensi yang dipasang akan terbuka.
3
Aktifkan Mode Pengembang
Di kanan atas halaman, cari tombol Mode Pengembang.

Kliknya untuk mengaktifkannya (menjadi biru).
💡 Jika Anda tidak melihatnya, gulir ke atas halaman.
4
Muat ekstensi
Setelah mengaktifkan Mode Pengembang, tombol baru akan muncul di atas.

Klik Muat ekstensi yang tidak dikemas (yang pertama di kiri).

Jendela untuk memilih folder akan terbuka.
5
Pilih folder AdOff
Navigasi ke folder yang diekstrak di Langkah 1.

Pilih adoff-chrome dan klik Pilih Folder (Windows/Linux) atau Buka (Mac).
✓ Selesai. AdOff dipasang. Anda akan melihat ikon di bilah Chrome.

1
Ekstrak file ZIP
1.1 Buka Finder (ikon smiley di Dock di bawah).
1.2 Di bilah kiri, klik Download.
1.3 Cari file adoff-safari.zip (yang baru saja Anda unduh).
1.4 Klik ganda file. macOS akan mengekstraknya secara otomatis dan membuat folder adoff-safari di sebelah ZIP.
1.5 Jangan pindahkan atau ubah nama folder ini. Biarkan di Download.
2
Pasang Xcode (gratis, ~10 GB)
2.1 Buka Mac App Store (ikon biru dengan "A" putih, di Dock atau Launchpad).
2.2 Di bilah pencarian di kiri atas, ketik Xcode dan tekan Return.
2.3 Klik Unduh atau Pasang di entri Xcode (ikon Apple, oleh Apple).
2.4 Masuk dengan Apple ID Anda jika diminta (sama dengan App Store).
2.5 Tunggu unduhan. Xcode besar (~10 GB), bisa memakan waktu 20-60 menit.
2.6 Ketika siap, buka Xcode setidaknya sekali: klik ikon di Launchpad atau Aplikasi.
2.7 Jendela dengan syarat lisensi muncul. Klik Terima.
2.8 Tunggu Xcode menyelesaikan penyiapan awal (1-2 menit), lalu tutup (Cmd+Q).
💡 Jika Mac Anda tidak memiliki ruang yang cukup, bebaskan setidaknya 15 GB sebelum memulai.
3
Buka Terminal dan jalankan perintah
3.1 Tekan Cmd + Space bersama. Bilah pencarian terbuka (Spotlight) di atas.
3.2 Ketik Terminal dan tekan Return. Jendela putih/hitam membuka (Terminal, tidak menakutkan, itu program Mac normal).
3.3 Salin perintah ini dengan tepat (klik kotak abu-abu untuk memilihnya, lalu Cmd+C):
xcrun safari-web-extension-converter ~/Downloads/adoff-safari
3.4 Klik di jendela Terminal, tempel dengan Cmd+V, lalu tekan Return.
3.5 Terminal bertanya "Continue?". Ketik y dan tekan Return.
3.6 Tunggu 30-60 detik. Xcode terbuka secara otomatis dengan proyek "AdOff".
3.7 Di Xcode, klik tombol ▶ Play di kiri atas (atau Cmd+R).
3.8 Kompilasi: tunggu 1-3 menit. Jendela kecil "AdOff" mungkin terbuka. Anda dapat menutupnya (ekstensi sudah dipasang).
⚠ Jika Anda mendapatkan "permission denied" atau "command not found": buka Xcode, Preferensi, Lokasi, Command Line Tools dan pilih versi terbaru. Lalu coba kembali langkah 3.4.
4
Aktifkan menu Pengembang di Safari
4.1 Buka Safari (ikon kompas di Dock).
4.2 Di bilah menu di atas, klik Safari → Preferensi (atau Cmd+,).
4.3 Jendela terbuka dengan beberapa tab di atas. Klik Lanjutan (yang terakhir, ikon roda gigi).
4.4 Di bawah, cari opsi Tampilkan fitur pengembangan web (atau "Tampilkan menu Pengembang" di versi lebih lama).
4.5 Centang kotak. Menu Pengembang baru muncul di bilah menu Safari.
4.6 Tutup jendela Preferensi.
5
Izinkan ekstensi yang tidak ditandatangani
5.1 Di bilah menu Safari, klik menu Pengembang baru.
5.2 Gulir menu dan klik Izinkan Ekstensi yang Tidak Ditandatangani.
5.3 Safari meminta kata sandi Mac Anda (yang Anda gunakan untuk masuk). Ketik dan tekan Return.
5.4 Opsi sekarang aktif (Anda dapat memverifikasi: membuka kembali menu Pengembang, "Izinkan Ekstensi yang Tidak Ditandatangani" memiliki tanda centang).
⚠ Penting: opsi ini menonaktifkan secara otomatis setiap kali Anda memulai ulang Safari. Anda harus mengaktifkannya kembali (langkah 5.1-5.3) setiap kali sistem dimulai ulang.
6
Aktifkan AdOff
6.1 Buka kembali Preferensi Safari: Safari → Preferensi (atau Cmd+,).
6.2 Klik tab Ekstensi (ikon puzzle).
6.3 Dalam daftar di sebelah kiri, Anda harus melihat AdOff. Centang kotak di sebelahnya.
6.4 Popup muncul yang menanyakan izin akses situs. Klik Izinkan Selalu di Semua Situs (atau "Izinkan di Semua Situs").
6.5 Tutup Preferensi. Kunjungi situs apa pun dengan iklan (berita, platform video, forum). Iklan diblokir.
6.6 Ikon AdOff (sebuah "A" dengan overlay merah) muncul secara otomatis di bilah Safari di kanan atas. Klik untuk membuka popup dengan kontrol.
✓ Selesai. AdOff aktif. Anda memiliki 30 hari uji coba Pro gratis. Ikon menunjukkan hitung mundur.

Pertanyaan yang Sering Diajukan

Apakah saya perlu menyimpan folder di komputer saya? ▼
Ya. Browser membaca file ekstensi dari folder. Jika Anda menghapusnya, ekstensi akan berhenti berfungsi. Letakkan di tempat yang aman (mis. Dokumen).
Apakah aman mengaktifkan Mode Pengembang? ▼
Ya, sepenuhnya aman. Mode Pengembang hanya memungkinkan Anda memasang ekstensi dari folder alih-alih dari toko. Tidak mengubah apa pun yang lain di browser.
Browser mengatakan "Nonaktifkan ekstensi pengembang" - apa yang harus saya lakukan? ▼
Beberapa browser menampilkan peringatan saat startup ketika Anda memiliki ekstensi yang dipasang secara manual. Cukup tutup peringatan. Ekstensi terus berfungsi secara normal. Ketika AdOff ada di toko resmi, peringatan ini akan hilang.
Bagaimana cara memperbarui AdOff? ▼
Unduh versi baru dari situs, ekstrak ZIP di folder yang sama (timpa file), lalu buka halaman ekstensi browser Anda dan klik panah "Muat Ulang" di AdOff.
Apakah itu bekerja di semua situs? ▼
Ya. AdOff memblokir iklan di semua situs web (spanduk, popup, pelacak, media sosial, berita, blog). Pemblokiran iklan video di platform streaming tersedia dalam Pro/Trial. Anda juga dapat menghentikan sementara pemblokiran di situs tertentu jika diperlukan.
Bagaimana cara menghapus instalan AdOff? ▼
Buka halaman ekstensi browser Anda, cari AdOff dan klik "Hapus". Kemudian Anda dapat menghapus folder dari komputer Anda.
Browser mana yang didukung? ▼
AdOff mendukung semua browser Chromium (Chrome, Edge, Brave, Opera, Vivaldi, Arc) dengan kompatibilitas fitur 100%, Firefox dengan 95%, dan Safari 16.4+ di macOS Ventura+ dengan 95% (memerlukan Xcode). Di Android gunakan Kiwi Browser atau Firefox. Dukungan iOS segera tiba.
Apakah itu bekerja di mobile? ▼
Di Android, pasang Kiwi Browser (gratis dari Play Store) yang mendukung ekstensi Chrome. Di iOS, ekstensi browser belum didukung. Kami sedang mengerjakannya.
