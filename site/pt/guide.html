🛠 1. O que é o AdOff

O AdOff é uma extensão para o Chrome que bloqueia publicidade em todos os sites. Opera em quatro níveis , rede, página, neutralização de anúncios em vídeo (apenas Pro/Trial) e stealth (apenas Pro/Trial) , para uma navegação limpa e rápida.

  • Bloqueia publicidade em qualquer site; neutralização de anúncios em vídeo nas plataformas de streaming (Pro/Trial)
  • Stealth Mode (Pro/Trial): torna o AdOff invisível aos sistemas anti-bloqueador
  • Ultraleve , conçu para ser l'extension la plus légère du marché
  • Não recolhe nenhum dado pessoal: tudo funciona localmente
  • 6 idiomas: italiano, inglês, alemão, francês, espanhol, português
  • Pausa por site com 4 durações: visita, 1 hora, 1 dia, permanente
  • Sistema de referral para ganhar dias Pro gratuitos

📥 2. Instalação

A partir da Chrome Web Store

  1. Vai à página do AdOff na Chrome Web Store
  2. Clica em "Adicionar ao Chrome"
  3. Confirma com "Adicionar extensão"
  4. O AdOff fica ativo de imediato

Fixar o ícone na barra de ferramentas

  1. Clica no ícone do puzzle (Extensões) no canto superior direito do browser
  2. Encontra o AdOff na lista
  3. Clica no alfinete junto ao AdOff
  4. O ícone aparece na barra de ferramentas para acesso rápido

🎉 3. Primeiro arranque

Quando instalas o AdOff pela primeira vez:

  • Abre-se uma página de boas-vindas com instruções para fixar o ícone
  • O Trial Pro de 30 dias ativa-se automaticamente
  • É gerado um código de referral único (formato: ADO-XXXXX)
  • O AdOff começa a bloquear publicidade de imediato
  • Se instalares antes de 1 de julho de 2026, recebes o badge Founding Member permanente

🖼 4. O Popup

Clica no ícone do AdOff na barra de ferramentas para abrir o popup.

Cabeçalho

Logótipo, nome "AdOff", badge de licença (FREE / TRIAL Xd / PRO) e badge FOUNDER (se aplicável).

Botão global

Interruptor ON/OFF para ativar ou desativar o AdOff em todos os sites. Badge roxo = ativo, vermelho = desativado.

Contadores

  • Anúncios bloqueados: elementos publicitários removidos das páginas
  • Pedidos bloqueados: pedidos HTTP a servidores de anúncios interceptados

Os números abreviam-se: 1.500 torna-se "1.5K", 2.300.000 torna-se "2.3M".

Site atual

O domínio do site que estás a visitar, com o botão "Pausar aqui".

Banner de licença

Visível apenas para utilizadores Free ("Experimenta o Pro 30 dias grátis") e Trial ("Trial ativo , X dias restantes"). Oculto para utilizadores Pro.

Pedido de avaliação

Aparece após 100+ anúncios bloqueados e 5+ dias desde a instalação. Máximo 3 aparições, com 14 dias de pausa entre cada uma. Três opções: "Deixar avaliação", "Mais tarde", "Nunca mais".

Changelog

Após uma atualização, mostra as novidades. Fecha-se com o X.

Links no fundo

Ajuda (abre a FAQ nas opções) e Opções (abre a página completa).

⏸ 5. Pausa por site

Desativa o AdOff num único site sem o desligar em todo o lado.

Como pausar

  1. Visita o site
  2. Clica no ícone do AdOff na barra de ferramentas
  3. Clica em "Pausar aqui"
  4. Escolhe a duração
OpçãoO que acontece
Apenas esta visitaReativa quando fechas o separador
1 horaReativa após 60 minutos
1 diaReativa após 24 horas
Sempre (lista branca)Fica desativado permanentemente

Como reativar

Visita o site em pausa, clica no ícone do AdOff, verás "Em pausa , [tipo]", clica em "Reativar".

Nota: as pausas expiradas são removidas automaticamente. O bloqueio efetivo funciona também para subdomínios: se pausares example.com, o blog.example.com também fica excluído.

🛡 7. Sites excluídos (Lista branca)

Adicionar um site

  1. Vai a Opções > Sites excluídos
  2. Escreve o domínio (ex. example.com)
  3. Clica em "Adicionar" ou pressiona Enter

Exportação e importação

Exportar: descarrega a lista branca como ficheiro JSON. Importar: carrega um ficheiro JSON. As entradas são validadas (domínio máx. 255 caracteres, tipo permitido). As inválidas são ignoradas.

⚠ 13. Reportar problemas

Se um site não funcionar com o AdOff, reporta-o em Opções > Ajuda > Reportar um problema.

  1. Insere o URL do site (obrigatório)
  2. Escolhe o tipo: Site não funciona / Anúncios visíveis / Anti-bloqueador / Outro
  3. Adiciona descrição e e-mail (opcionais)
  4. Resolve o captcha matemático
  5. Clica em "Enviar relatório"

O formulário inclui proteções anti-bot: honeypot, temporização, captcha, rate limit (máx. 3/hora), anti-spam, deduplicação.

⚙ 15. Como funciona o AdOff (técnico)

Nível 1 , Bloqueio de rede

O AdOff usa a API declarativeNetRequest do Chrome para bloquear os pedidos HTTP a servidores de anúncios antes de o browser os descarregar. Mais de 120 regras cobrem as principais redes de anúncios e muitas outras.

Nível 2 , Filtragem cosmética

Para publicidade servida pelo mesmo domínio do site, o AdOff usa CSS e JavaScript para ocultá-la da página: seletores precisos, MutationObserver para conteúdo dinâmico, colapso de contentores vazios.

Nível 3 , Neutralização de anúncios em vídeo (apenas Pro/Trial) PRO

A maioria dos players de vídeo usa um SDK de publicidade padrão para as interrupções (pre-roll, mid-roll, post-roll). O AdOff substitui-o por uma versão neutra que comunica imediatamente ao player "sem publicidade" , o vídeo inicia sem interrupções. Funciona automaticamente em qualquer site do mundo. Nas principais plataformas de vídeo: aceleração a 16x + clique automático em "Ignorar anúncio". Funcionalidade exclusiva Pro.

Nível 4 , Stealth Mode (apenas Pro/Trial)

Interceta as verificações anti-bloqueador e faz o site acreditar que tudo está visível:

  • Spoofing CSS: propriedades dos elementos de isca falsificadas
  • Intercetação de rede: scripts de deteção recebem respostas vazias
  • Variáveis simuladas: SDKs de publicidade pré-preenchidos
  • Neutralização de scripts: scripts anti-bloqueador bloqueados antes da execução
  • Anti scroll-lock: impede que muros bloqueiem o scroll

O stealth é desativado automaticamente nas principais plataformas (vídeo, redes sociais, e-commerce) e outros sites grandes onde interferiria.

🛠 16. Resolução de problemas

Um site não funciona

  1. Clica no ícone do AdOff e usa "Pausar aqui"
  2. Recarrega a página
  3. Se funcionar em pausa: reporta-nos o site
  4. Se não funcionar mesmo em pausa: o problema não é do AdOff

Ainda vejo publicidade

  1. Verifica que o AdOff está ativo (badge roxo)
  2. Verifica que o site não está na lista branca
  3. Recarrega a página
  4. Se persistir: reporta-nos o site

"Desativa o teu bloqueador de anúncios"

  1. Recarrega a página
  2. Verifica que tens Pro ou Trial (é necessário o Stealth Mode)
  3. Se continuar: usa "Pausar aqui" e reporta-nos

O AdOff não ativa

Vai a chrome://extensions/, verifica que está ativado. Se houver erros, tenta remover e reinstalar.

🔒 17. Privacidade e segurança

O AdOff NÃO recolhe dados pessoais, NÃO rastreia os sites que visitas, NÃO envia telemetria, NÃO partilha dados com terceiros, NÃO injeta publicidade.

Comunicações externas

O AdOff comunica com servidores externos apenas por ação tua: reportar site, sugestão, ativação de licença, revalidação automática (a cada 7 dias para Pro). As comunicações passam por Cloudflare Worker.

🗑 19. Desinstalar

  1. Vai a chrome://extensions/
  2. Encontra o AdOff e clica em "Remover"
  3. Confirma

As definições locais são apagadas. A licença Pro mantém-se válida no servidor. O código de referral é perdido.

Antes de desinstalar, exporta as definições em Opções > Avançadas > Exportar. Após reinstalação, importa o ficheiro para restaurar tudo (a licença tem de ser reativada com a chave).
