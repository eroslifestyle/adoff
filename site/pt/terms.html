2. Descrição do Serviço

O AdOff é uma extensão para o browser Chrome que bloqueia publicidade ao nível da rede e visual, e impede que os sistemas de deteção de anti-bloqueadores identifiquem a extensão. O Serviço é composto por:

  • A Extensão AdOff para Chrome , o software instalado no teu browser

Os utilizadores podem criar opcionalmente uma conta AdOff para gerir os seus dispositivos e licenças a partir de um painel único em adoff.app/account. A criação de conta não é obrigatória para nenhum nível, incluindo Pro.

Os dados da conta são regidos pela nossa Política de privacidade. A versão gratuita não requer conta e não envolve comunicação com o servidor.

Cada instalação de browser conta como um dispositivo. Os utilizadores Pro podem ver e gerir os seus dispositivos ativos em adoff.app/account. Para permanecer dentro do limite de 3 dispositivos, os utilizadores podem remover dispositivos no painel da conta. Aplica-se um máximo de 3 remoções de dispositivos por dia para prevenir abusos.
