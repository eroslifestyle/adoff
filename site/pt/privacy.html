Política de privacidade

Última atualização: 21 de abril de 2026  ·  Versão 2.0

Em resumo: AdOff não recolhe dados de navegação pessoais. Não te rastreia. Não vende dados a ninguém. A única comunicação com os nossos servidores é a validação da licença Pro , e mesmo aí, partilhamos apenas o mínimo indispensável. A versão GRATUITA funciona sem registo nem conta , não são recolhidos dados dos utilizadores gratuitos para além do que é armazenado localmente no dispositivo.

1. Quem somos

AdOff é uma extensão para browser que bloqueia publicidade e rastreadores. O responsável pelo tratamento de dados é a equipa AdOff, contactável através da página de suporte.

2. Dados que não recolhemos

AdOff não recolhe, não transmite e não conserva:

  • Histórico de navegação ou URLs dos sites visitados
  • Conteúdo de páginas ou dados introduzidos em formulários
  • Cookies de perfilagem ou rastreamento de terceiros
  • Dados comportamentais ou de perfilagem
  • Quaisquer dados que possam ser usados para te perfilar sem o teu consentimento

A versão GRATUITA funciona sem registo nem conta. Não são recolhidos dados dos utilizadores gratuitos para além do que é armazenado localmente no dispositivo.

3. Dados da conta (apenas utilizadores Pro)

Se criares uma conta ou adquirires um plano Pro, recolhemos os seguintes dados:

  • Endereço de e-mail , recolhido na compra (através do Stripe) e no registo de conta. Utilizado para: entrega de licença, acesso à conta, comunicações de serviço. Base jurídica: execução do contrato (Art. 6.º(1)(b) RGPD).
  • Palavra-passe , armazenada como hash irreversível (SHA-256 + sal). Nunca em texto simples. Nunca partilhada com terceiros.
  • Google OAuth , se usares "Iniciar sessão com Google", recebemos o teu e-mail e um ID de perfil do Google. Não acedemos a quaisquer outros dados da tua conta Google.
  • UUID do dispositivo , um identificador aleatório gerado na instalação da extensão. Usado para aplicar os limites de dispositivos por licença. Não está associado à tua identidade. Base jurídica: interesse legítimo (Art. 6.º(1)(f) RGPD).

4. Dados armazenados localmente

AdOff usa chrome.storage.local e chrome.storage.sync para guardar localmente no teu dispositivo:

  • A chave de licença Pro (se adquirida)
  • As preferências da extensão (ex. lista branca de sites, definições da interface)
  • O estado de ativação (ativo/inativo por site)

Estes dados nunca são enviados para servidores de terceiros e permanecem no teu browser. chrome.storage.sync pode sincronizar dados entre dispositivos através da tua conta Google, de acordo com as políticas da Google.

5. Validação de licença

Se adquiriste um plano Pro, a extensão contacta os nossos servidores para verificar a validade da licença. Nessa comunicação são transmitidos:

  • A chave de licença
  • O UUID do dispositivo
  • Identificador de browser/SO (para o painel de gestão de dispositivos)
  • A versão da extensão

O teu endereço IP é visível para o nosso servidor (Cloudflare) mas não é armazenado permanentemente , apenas é derivada uma geolocalização ao nível país/cidade para análises agregadas. Os registos de acesso ao servidor são eliminados no prazo de 30 dias.

6. Terceiros

AdOff não partilha, não vende nem cede dados a terceiros para fins de marketing ou perfilagem. Não utilizamos:

  • Serviços de análise comportamental
  • Serviços de publicidade
  • SDK de rastreamento de terceiros

Serviços de terceiros que utilizamos para prestar o serviço:

  • Stripe , processamento de pagamentos. Os dados de pagamento são geridos pelo Stripe de acordo com a sua própria Política de privacidade.
  • Google Sign-In , se usares "Iniciar sessão com Google", a tua autenticação é gerida pelo Google. Consulta a política de privacidade da Google.
  • Cloudflare , alojamento e distribuição de conteúdos. O Cloudflare pode processar endereços IP de acordo com a sua política de privacidade.

7. Permissões da extensão

AdOff requer as seguintes permissões do browser, todas necessárias para o seu funcionamento:

  • declarativeNetRequest , para bloquear pedidos de rede (anúncios, rastreadores)
  • storage , para guardar preferências e licença localmente
  • tabs , para aplicar as definições ao separador ativo
  • scripting , para injetar CSS de ocultação em páginas web
  • webNavigation , para reaplicar os filtros ao navegar entre páginas no mesmo separador

Nenhuma destas permissões é usada para recolher ou transmitir dados de navegação.

8. Os teus direitos (RGPD)

Ao abrigo do Regulamento Geral sobre a Proteção de Dados da UE, tens direito a:

  • Acesso , solicitar uma cópia de quaisquer dados que conservemos sobre ti
  • Retificação , solicitar a correção de dados inexatos
  • Apagamento , solicitar a eliminação dos teus dados («direito a ser esquecido»)
  • Portabilidade de dados , receber os teus dados num formato portátil
  • Limitação , solicitar a limitação do tratamento
  • Oposição , opor-te ao tratamento dos teus dados
  • Reclamação , apresentar uma queixa junto de uma autoridade de controlo na União Europeia
  • Eliminação de conta , eliminar a tua conta e todos os dados associados
  • Exportação de dados , exportar os dados da tua conta
  • Gestão de dispositivos , gerir os teus dispositivos com licença em adoff.app/account

Para exercer os teus direitos, contacta-nos através da página de suporte.

9. Retenção de dados

  • Preferências locais: armazenadas no teu dispositivo até desinstalar a extensão ou limpar os dados do browser.
  • Dados da conta (e-mail, palavra-passe com hash): conservados enquanto a conta estiver ativa. Eliminados a pedido.
  • UUID do dispositivo: removido quando retiras o dispositivo do teu painel de conta.
  • Geolocalização derivada do IP: agregada apenas ao nível país. Nenhum endereço IP individual é armazenado permanentemente.
  • Palavras-passe com hash: eliminadas quando a conta é apagada.
  • Registos de validação de licença: eliminados automaticamente no prazo de 30 dias.
  • Dados de pagamento: geridos pelo nosso processador de pagamentos de acordo com as suas políticas de retenção.

10. Base jurídica para o tratamento

A base jurídica para o tratamento dos dados de conta e licença é a execução do contrato (Art. 6.º(1)(b) RGPD) , necessário para verificar a tua licença Pro e prestar o serviço que adquiriste. O tratamento do UUID do dispositivo baseia-se no interesse legítimo (Art. 6.º(1)(f) RGPD) , necessário para aplicar os limites de dispositivos por licença.

11. Menores

AdOff não se destina a pessoas com menos de 13 anos. Não recolhemos conscientemente dados de menores.

12. Alterações a esta política

Poderemos atualizar esta Política de privacidade no futuro. Quaisquer alterações significativas serão comunicadas através da extensão ou do site. Ao continuares a usar AdOff após uma alteração, aceitas a nova versão da política.

14. Dados Diagnósticos Voluntários

O AdOff inclui uma função de relatório de vazamento de anúncios opt-in: quando a extensão detecta que um anúncio está visível numa página e escolhes reportá-lo, um pacote de diagnóstico é enviado aos nossos servidores para análise. Esta função é completamente opcional e desativada por padrão. Nenhum dado de diagnóstico é enviado sem o teu consentimento explícito. Quando envias um relatório, são transmitidas as seguintes informações:

  • O URL da página onde o anúncio foi detectado
  • A versão da extensão AdOff instalada
  • Dados técnicos sobre o elemento publicitário visível (ex. seletor CSS, posição)
  • A lista de regras de filtro ativas (não o teu histórico de navegação)
  • Informações básicas do navegador e sistema operativo
Estes dados são utilizados exclusivamente para melhorar as regras de bloqueio de anúncios. Não são associados à tua identidade, vendidos ou partilhados com terceiros. Os relatórios de diagnóstico são mantidos por um máximo de 90 dias e depois eliminados.
