🛠 1. Qu'est-ce qu'AdOff

AdOff est une extension pour Chrome qui bloque les publicités sur tous les sites web. Il fonctionne sur quatre niveaux , réseau, page, neutralisation des publicités vidéo (Pro/Essai uniquement) et mode furtif (Pro/Essai uniquement) , pour une navigation propre et rapide.

  • Bloque les publicités sur n'importe quel site ; neutralisation des publicités vidéo sur les plateformes de streaming (Pro/Essai)
  • Mode Furtif (Pro/Essai) : rend AdOff invisible aux systèmes anti-adblock
  • Ultraléger , conçu pour être l'extension la plus légère du marché
  • Ne collecte aucune donnée personnelle : tout fonctionne en local
  • 6 langues : italien, anglais, allemand, français, espagnol, portugais
  • Pause par site avec 4 durées : visite, 1 heure, 1 jour, permanente
  • Système de parrainage pour obtenir des jours Pro gratuits

📥 2. Installation

Depuis le Chrome Web Store

  1. Rendez-vous sur la page d'AdOff dans le Chrome Web Store
  2. Cliquez sur "Ajouter à Chrome"
  3. Confirmez avec "Ajouter l'extension"
  4. AdOff est actif immédiatement

Épingler l'icône dans la barre d'outils

  1. Cliquez sur l'icône puzzle (Extensions) en haut à droite du navigateur
  2. Trouvez AdOff dans la liste
  3. Cliquez sur l'épingle à côté d'AdOff
  4. L'icône apparaît dans la barre d'outils pour un accès rapide

🎉 3. Premier démarrage

Lors de la première installation d'AdOff :

  • Une page de bienvenue s'ouvre avec des instructions pour épingler l'icône
  • L'Essai Pro de 30 jours s'active automatiquement
  • Un code de parrainage unique est généré (format : ADO-XXXXX)
  • AdOff commence à bloquer les publicités immédiatement
  • Si vous installez avant le 1er juillet 2026, vous recevez le badge Membre Fondateur permanent

🖼 4. Le Popup

Cliquez sur l'icône AdOff dans la barre d'outils pour ouvrir le popup.

En-tête

Logo, nom "AdOff", badge de licence (FREE / ESSAI Xjr / PRO) et badge FONDATEUR (si vous y avez droit).

Interrupteur global

Bouton ON/OFF pour activer ou désactiver AdOff sur tous les sites. Badge violet = actif, rouge = désactivé.

Compteurs

  • Publicités bloquées : éléments publicitaires supprimés des pages
  • Requêtes bloquées : requêtes HTTP vers des serveurs publicitaires interceptées

Les chiffres s'abrègent : 1 500 devient "1,5K", 2 300 000 devient "2,3M".

Site actuel

Le domaine du site que vous visitez, avec le bouton "Pause ici".

Bannière de licence

Visible uniquement pour les utilisateurs Free ("Essayez Pro 30 jours gratuits") et Essai ("Essai actif , X jours restants"). Masquée pour les utilisateurs Pro.

Invite d'avis

Apparaît après 100+ publicités bloquées et 5+ jours depuis l'installation. Maximum 3 apparitions, avec 14 jours de pause entre chaque. Trois options : "Laisser un avis", "Pas maintenant", "Ne plus demander".

Journal des modifications

Après une mise à jour, affiche les nouveautés. Se ferme avec le X.

Liens en bas

Aide (ouvre la FAQ dans les options) et Options (ouvre la page complète).

⏸ 5. Pause par site

Désactivez AdOff sur un seul site sans l'éteindre partout.

Comment mettre en pause

  1. Visitez le site
  2. Cliquez sur l'icône AdOff dans la barre d'outils
  3. Cliquez sur "Pause ici"
  4. Choisissez la durée
OptionCe qui se passe
Cette visite seulementSe réactive quand vous fermez l'onglet
1 heureSe réactive après 60 minutes
1 jourSe réactive après 24 heures
Toujours (liste blanche)Reste désactivé de façon permanente

Comment réactiver

Visitez le site en pause, cliquez sur l'icône AdOff, vous verrez "En pause , [type]", cliquez sur "Réactiver".

Remarque : les pauses expirées sont supprimées automatiquement. Le blocage effectif fonctionne aussi pour les sous-domaines : si vous mettez en pause example.com, blog.example.com est également exclu.

🛡 7. Sites exclus (Liste blanche)

Ajouter un site

  1. Allez dans Options > Sites exclus
  2. Saisissez le domaine (ex. example.com)
  3. Cliquez sur "Ajouter" ou appuyez sur Entrée

Exportation et importation

Exporter : télécharge la liste blanche en fichier JSON. Importer : charge un fichier JSON. Les entrées sont validées (domaine max 255 caractères, type autorisé). Les entrées invalides sont ignorées.

⚠ 13. Signalement de problèmes

Si un site ne fonctionne pas avec AdOff, signalez-le dans Options > Aide > Signaler un problème.

  1. Saisissez l'URL du site (obligatoire)
  2. Choisissez le type : Site ne fonctionne pas / Publicités visibles / Anti-adblock / Autre
  3. Ajoutez une description et un e-mail (facultatifs)
  4. Résolvez le captcha mathématique
  5. Cliquez sur "Envoyer le signalement"

Le formulaire inclut des protections anti-bot : honeypot, timing, captcha, limite de débit (max 3/heure), anti-spam, déduplication.

💡 14. Suggestions

Dans Options > Suggestions, vous pouvez proposer des fonctionnalités, signaler des bugs ou des améliorations. Remplissez le titre (obligatoire), la description et l'e-mail (facultatifs). Vos suggestions apparaissent dans une liste sous le formulaire.

⚙ 15. Comment fonctionne AdOff (technique)

Niveau 1 , Blocage réseau

AdOff utilise l'API declarativeNetRequest de Chrome pour bloquer les requêtes HTTP vers les serveurs publicitaires avant que le navigateur ne les télécharge. Plus de 120 règles couvrent les principaux réseaux publicitaires et de nombreux autres.

Niveau 2 , Filtrage cosmétique

Pour les publicités servies depuis le même domaine que le site, AdOff utilise CSS et JavaScript pour les masquer depuis la page : sélecteurs précis, MutationObserver pour les contenus dynamiques, réduction des conteneurs vides.

Niveau 3 , Neutralisation des publicités vidéo (Pro/Essai uniquement) PRO

La plupart des lecteurs vidéo utilisent un SDK publicitaire standard pour les interruptions (pre-roll, mid-roll, post-roll). AdOff le remplace par une version neutre qui communique immédiatement au lecteur "aucune publicité" , la vidéo démarre sans interruption. Fonctionne automatiquement sur n'importe quel site dans le monde. Sur les principales plateformes vidéo : accélération à 16x + clic automatique "Passer l'annonce". Fonctionnalité exclusive Pro.

Niveau 4 , Mode Furtif (Pro/Essai uniquement)

Intercepte les vérifications anti-adblock et fait croire au site que tout est visible :

  • Usurpation CSS : propriétés des éléments leurres falsifiées
  • Interception réseau : les scripts de détection reçoivent des réponses vides
  • Variables simulées : SDK publicitaires pré-remplis
  • Neutralisation de scripts : scripts anti-adblock bloqués avant exécution
  • Anti-verrouillage de défilement : empêche les murs de bloquer le défilement

Le mode furtif est désactivé automatiquement sur les principales plateformes (vidéo, réseaux sociaux, e-commerce) et d'autres grands sites où il interfèrerait.

🛠 16. Résolution de problèmes

Un site ne fonctionne pas

  1. Cliquez sur l'icône AdOff et utilisez "Pause ici"
  2. Rechargez la page
  3. Si ça fonctionne en pause : signalez-nous le site
  4. Si ça ne fonctionne pas non plus en pause : le problème ne vient pas d'AdOff

Je vois encore des publicités

  1. Vérifiez qu'AdOff est actif (badge violet)
  2. Vérifiez que le site n'est pas dans la liste blanche
  3. Rechargez la page
  4. Si ça persiste : signalez-nous le site

"Désactivez votre bloqueur de publicités"

  1. Rechargez la page
  2. Vérifiez que vous avez Pro ou Essai (le Mode Furtif est nécessaire)
  3. Si ça continue : utilisez "Pause ici" et signalez-le nous

AdOff ne s'active pas

Allez sur chrome://extensions/, vérifiez qu'il est activé. S'il y a des erreurs, essayez de le désinstaller et de le réinstaller.

🗑 19. Désinstallation

  1. Allez sur chrome://extensions/
  2. Trouvez AdOff et cliquez sur "Supprimer"
  3. Confirmez

Les paramètres locaux sont supprimés. La licence Pro reste valide sur le serveur. Le code de parrainage est perdu.

Avant de désinstaller, exportez les paramètres depuis Options > Avancé > Exporter. Après la réinstallation, importez le fichier pour tout restaurer (la licence doit être réactivée avec la clé).
