How to Contribute

AdOff is shaped by its community. Every contribution, big or small, makes a difference.

🐛
Report a Bug
Found an ad that slipped through? A broken site? Open a report on GitHub or Discord and we'll investigate. Include the site URL and steps to reproduce.
Open a Report →
💡
Suggest a Feature
Have an idea to improve AdOff? Share it in GitHub Discussions or Discord. The best ideas make it into the next release.
Share Your Idea →
⭐
Write a Review
A review on the Chrome Web Store helps other users discover AdOff. Takes 30 seconds and makes a huge difference for an independent project like ours.
Leave a Review →
🎁
Share with Friends
Use your personal referral link from AdOff Options to invite friends. When they go Pro, you both earn free Pro days with no limit on how many you can earn.
Get Your Link →
🌍
Translate AdOff
AdOff currently supports 15 languages. Want to add yours or improve an existing translation? Contact us on Discord. Translation contributors receive a free Pro license.
Help Translate →
📝
Contribute Filter Rules
If you're technical, help us improve our 107+ blocking rules. Submit rule suggestions via GitHub and help catch ads slipping through on niche sites.
