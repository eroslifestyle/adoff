The fastest way: install an ad blocker

Chrome doesn't block ads on its own. You add the capability with an extension. Here's the whole process:

  1. Open the Chrome Web Store and search for your ad blocker (or go straight to the developer's install page).
  2. Click Add to Chrome, then confirm.
  3. Pin the extension (puzzle icon → pin) so you can pause it per-site when you want to support a creator.
  4. That's it. Reload any page and the ads are gone.
