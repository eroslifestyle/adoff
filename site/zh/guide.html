🛠 1. 什么是 AdOff

AdOff 是一款适用于 Google Chrome 的扩展程序，可在所有网站上拦截广告。它从四个层面运作——网络层、页面层、视频广告中性化（仅限 Pro/试用版）和隐身模式（仅限 Pro/试用版）——让您获得干净、快速的浏览体验。

  • 拦截任何网站上的广告；在流媒体平台上中性化视频广告 （Pro/试用版）
  • 隐身模式（Pro/试用版）：让 AdOff 对反广告拦截系统不可见
  • 超轻量 — 原生 Manifest V3 设计
  • 不收集任何个人数据：一切在本地运行
  • 支持 6 种语言：意大利语、英语、德语、法语、西班牙语、葡萄牙语
  • 网站暂停功能，提供 4 种时长：本次访问、1小时、1天、永久
  • 推荐系统，可免费获得 Pro 天数

📥 2. 安装

从 Chrome 网上应用店安装

  1. 前往 Chrome 网上应用店中的 AdOff 页面
  2. 点击 "添加到 Chrome"
  3. 点击 "添加扩展程序" 确认
  4. AdOff 立即生效

将图标固定到工具栏

  1. 点击浏览器右上角的 拼图 图标（扩展程序）
  2. 在列表中找到 AdOff
  3. 点击 AdOff 旁边的 图钉 图标
  4. 图标将显示在工具栏中，方便快速访问

⏸ 5. 网站暂停

在单个网站上停用 AdOff，而不影响其他网站。

如何暂停

  1. 访问该网站
  2. 点击工具栏中的 AdOff 图标
  3. 点击 "在此暂停"
  4. 选择时长
选项效果
仅本次访问关闭标签页时重新激活
1小时60分钟后重新激活
1天24小时后重新激活
始终（白名单）永久停用

如何重新激活

访问已暂停的网站，点击 AdOff 图标，您将看到"已暂停 — [类型]"，点击 "重新激活"。

注意：过期的暂停会自动移除。拦截对子域名同样有效：如果您暂停 example.com，则 blog.example.com 也会被排除。

⚙ 15. AdOff 工作原理（技术说明）

第1层 — 网络拦截

AdOff 使用 Chrome 的 declarativeNetRequest API，在浏览器下载广告之前就拦截对广告服务器的 HTTP 请求。超过 120 条规则涵盖主要广告网络和众多其他广告网络。

第2层 — 视觉过滤

对于由网站自身域名提供的广告，AdOff 使用 CSS 和 JavaScript 将其从页面中隐藏：精确选择器、用于动态内容的 MutationObserver、折叠空容器。

第3层 — 视频广告中性化（仅限 Pro/试用版）PRO

大多数视频播放器使用标准广告 SDK 进行插播广告（前贴片、中贴片、后贴片）。AdOff 用一个中性版本替换该 SDK，立即告知播放器"没有广告"——视频无中断播放。适用于全球任何网站。在主要视频平台上：加速至 16 倍 + 自动点击"跳过广告"。Pro 专属功能。

第4层 — 隐身模式（仅限 Pro/试用版）

拦截反广告拦截检测，让网站相信一切正常可见：

  • CSS 欺骗：诱饵元素属性被伪造
  • 网络拦截：检测脚本收到空响应
  • 变量模拟：预填充广告 SDK
  • 脚本中性化：反广告拦截脚本在执行前被阻止
  • 反滚动锁定：阻止广告墙锁定页面滚动

隐身模式在主要平台（视频、社交媒体、电商）和其他可能产生干扰的大型网站上自动停用。

🛠 16. 故障排查

某个网站无法使用

  1. 点击 AdOff 图标并使用 "在此暂停"
  2. 重新加载页面
  3. 如果暂停后正常：请告知我们该网站
  4. 如果暂停后仍不正常：问题与 AdOff 无关

仍然看到广告

  1. 确认 AdOff 已启用（紫色徽章）
  2. 确认该网站不在 白名单 中
  3. 重新加载页面
  4. 如果问题持续：请告知我们该网站

"请关闭您的广告拦截器"

  1. 重新加载页面
  2. 确认您拥有 Pro 或试用版（需要隐身模式）
  3. 如果继续出现：使用"在此暂停"并告知我们

AdOff 无法激活

前往 chrome://extensions/，确认已启用。如果出现错误，尝试移除并重新安装。

🔒 17. 隐私与安全

AdOff 不收集个人数据，不追踪您访问的网站，不发送遥测数据，不与第三方共享数据，不注入广告。

权限及其用途

权限用途
storage在本地保存设置和计数器
declarativeNetRequest拦截对广告网络的请求
activeTab了解域名以暂停特定网站
scripting注入内容脚本以隐藏广告
webNavigation在适当时机激活拦截
<all_urls>在所有网站上运行

外部通信

AdOff 仅在您主动操作时才与外部服务器通信：网站反馈、建议、许可证激活、自动重新验证（Pro 用户每 7 天一次）。通信通过 Cloudflare Worker 进行。

🗑 19. 卸载

  1. 前往 chrome://extensions/
  2. 找到 AdOff 并点击 "移除"
  3. 确认

本地设置将被删除。Pro 许可证在服务器上仍然有效。推荐码将丢失。

卸载前，请从"选项 > 高级 > 导出"导出设置。重新安装后，导入文件以恢复所有内容（许可证需要用密钥重新激活）。
