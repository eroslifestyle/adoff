🛠 1. AdOffとは

AdOffはGoogle Chrome向けの拡張機能で、すべてのウェブサイト上の広告をブロックします。ネットワーク、ページ、動画広告の無効化（Pro/Trialのみ）、ステルス（Pro/Trialのみ）の4つのレベルで動作し、クリーンで高速なブラウジングを実現します。

  • あらゆるサイトの広告をブロック；動画ストリーミングプラットフォームの動画広告を無効化 （Pro/Trial）
  • ステルスモード（Pro/Trial）：アドブロック対策サイトからAdOffを見えなくします
  • 超軽量 — ネイティブ Manifest V3 設計
  • 個人データを一切収集しません：すべてがローカルで動作します
  • 6か国語：イタリア語、英語、ドイツ語、フランス語、スペイン語、ポルトガル語
  • 4つの期間でサイトを一時停止：この訪問のみ、1時間、1日、永続
  • 無料のProデイを獲得できる紹介システム

📥 2. インストール

Chrome ウェブストアから

  1. Chrome ウェブストアのAdOffページに移動します
  2. 「Chromeに追加」をクリックします
  3. 「拡張機能を追加」で確認します
  4. AdOffがすぐに有効になります

ツールバーにアイコンをピン留め

  1. ブラウザ右上のパズルピースアイコン（拡張機能）をクリックします
  2. リストから AdOff を見つけます
  3. AdOffの横にあるピンアイコンをクリックします
  4. アイコンがツールバーに表示され、クイックアクセスが可能になります

🎉 3. 初回起動

AdOffを初めてインストールすると：

  • アイコンをピン留めする手順を示すウェルカムページが開きます
  • 15日間のPro無料トライアルが自動的に有効になります
  • ユニークな紹介コードが生成されます（形式：ADO-XXXXX）
  • AdOffがすぐに広告のブロックを開始します
  • 2026年7月1日以前にインストールすると、永続的な創設メンバーバッジを受け取ります

🖼 4. ポップアップ

ツールバーのAdOffアイコンをクリックしてポップアップを開きます。

ヘッダー

ロゴ、「AdOff」という名前、ライセンスバッジ（FREE / TRIAL X日 / PRO）と FOUNDER バッジ（対象の場合）。

グローバルトグル

すべてのサイトでAdOffを有効・無効にするON/OFFスイッチ。紫バッジ = 有効、赤バッジ = 無効。

カウンター

  • ブロックされた広告：ページから削除された広告要素
  • ブロックされたリクエスト：傍受された広告サーバーへのHTTPリクエスト

数値は省略されます：1,500は「1.5K」、2,300,000は「2.3M」になります。

現在のサイト

現在訪問しているサイトのドメイン、「ここで一時停止」ボタン付き。

ライセンスバナー

Freeユーザー（「15日間無料Proトライアル」）とTrialユーザー（「トライアル有効中 — あとX日」）にのみ表示されます。Proユーザーには非表示。

レビュープロンプト

100件以上の広告をブロックし、インストールから5日以上経過した後に表示されます。最大3回表示され、間隔は14日です。3つの選択肢：「レビューを書く」「後で」「二度と表示しない」。

変更履歴

アップデート後、新機能を表示します。Xボタンで閉じます。

フッターリンク

ヘルプ（オプションのFAQを開く）とオプション（完全なページを開く）。

⏸ 5. サイト一時停止

AdOff全体を無効にせずに、単一のサイトでAdOffを無効にします。

一時停止する方法

  1. サイトを訪問します
  2. ツールバーのAdOffアイコンをクリックします
  3. 「ここで一時停止」をクリックします
  4. 期間を選択します
オプション動作内容
この訪問のみタブを閉じると再有効化されます
1時間60分後に再有効化されます
1日24時間後に再有効化されます
常に（ホワイトリスト）永続的に無効のままになります

再有効化する方法

一時停止中のサイトを訪問し、AdOffアイコンをクリックすると「一時停止中 — [タイプ]」と表示されます。「再有効化」をクリックします。

注意：期限切れの一時停止は自動的に削除されます。ブロックはサブドメインにも有効です：example.comを一時停止すると、blog.example.comも除外されます。

🛡 7. 除外サイト（ホワイトリスト）

サイトを追加する

  1. オプション > 除外サイトに移動します
  2. ドメインを入力します（例：example.com）
  3. 「追加」をクリックするかEnterキーを押します

エクスポートとインポート

エクスポート：ホワイトリストをJSONファイルとしてダウンロードします。インポート：JSONファイルを読み込みます。エントリは検証されます（ドメイン最大255文字、許可されたタイプ）。無効なものは無視されます。

🎁 9. 紹介システム

友達を招待して無料のProデイを獲得しましょう。

  1. 初回インストール時にユニークコードを受け取ります：ADO-XXXXX
  2. リンク adoff.app/r/あなたのコード を共有します
  3. 友達がProに移行すると、+15日間の無料Proを獲得します
  4. 日数は無制限に蓄積されます

オプション > 友達を招待でリンク、クイック共有（WhatsApp、Telegram、メール）、紹介ダッシュボードを確認できます。

📊 10. 統計

オプション > 統計では、ブロックされた広告と傍受されたリクエストの合計カウンターを確認できます。「統計をリセット」で消去できます（確認が必要）。

🔧 11. 詳細設定

  • 設定のエクスポート：完全なJSONバックアップ（セキュリティのためライセンスデータを除く）
  • 設定のインポート：バックアップから復元します。ライセンスフィールドは自動的に無視されます
  • すべてをリセット：ライセンスを含むすべてを削除します。元に戻せません。二重確認が必要です

⚠ 13. 問題報告

AdOffでサイトが機能しない場合は、オプション > ヘルプ > 問題を報告から報告してください。

  1. サイトのURLを入力します（必須）
  2. タイプを選択します：サイトが動かない / 広告が表示される / アドブロック検出 / その他
  3. 説明とメールアドレスを追加します（任意）
  4. 数学的なキャプチャを解きます
  5. 「報告を送信」をクリックします

フォームにはボット対策が含まれています：ハニーポット、タイミング、キャプチャ、レート制限（最大3回/時間）、スパム対策、重複排除。

💡 14. 提案

オプション > 提案では、機能の提案、バグの報告、改善案を送ることができます。タイトル（必須）、説明、メールアドレス（任意）を入力します。あなたの提案はフォームの下のリストに表示されます。

⚙ 15. AdOffの仕組み（技術的説明）

レベル1 — ネットワークブロック

AdOffはChromeの declarativeNetRequest APIを使用して、ブラウザがダウンロードする前に広告サーバーへのHTTPリクエストをブロックします。120以上のルールが主要な広告ネットワークとその他多くの広告ネットワークをカバーします。

レベル2 — コスメティックフィルタリング

サイト自身のドメインから配信される広告については、AdOffはCSSとJavaScriptを使用してページから非表示にします：精密なセレクター、動的コンテンツ用のMutationObserver、空のコンテナの折りたたみ。

レベル3 — 動画広告の無効化（Pro/Trialのみ）PRO

ほとんどの動画プレイヤーは、割り込み広告（プレロール、ミッドロール、ポストロール）に標準的な広告SDKを使用しています。AdOffはそれをニュートラルなバージョンに置き換え、プレイヤーに即座に「広告なし」を伝えます — 動画が広告なしで開始されます。世界中のあらゆるサイトで自動的に機能します。主要な動画プラットフォームでは：16倍速 + 「広告をスキップ」の自動クリック。Pro限定機能。

レベル4 — ステルスモード（Pro/Trialのみ）

アドブロック対策の検出をインターセプトし、すべてが見えているかのようにサイトに信じ込ませます：

  • CSSスプーフィング：おとり要素のプロパティを偽装
  • ネットワークインターセプション：検出スクリプトが空のレスポンスを受け取る
  • 変数シミュレーション：広告SDKを事前に設定
  • スクリプトの無効化：アドブロック対策スクリプトを実行前にブロック
  • スクロールロック対策：ウォールがスクロールをブロックするのを防止

ステルスは主要なプラットフォーム（動画、ソーシャル、Eコマース）や干渉する可能性のある他の大規模サイトでは自動的に無効になります。

🛠 16. トラブルシューティング

サイトが機能しない

  1. AdOffアイコンをクリックして「ここで一時停止」を使用します
  2. ページをリロードします
  3. 一時停止中に機能する場合：サイトを報告してください
  4. 一時停止中も機能しない場合：問題はAdOffにはありません

まだ広告が表示される

  1. AdOffが有効であることを確認します（紫バッジ）
  2. サイトがホワイトリストにないことを確認します
  3. ページをリロードします
  4. 続く場合：サイトを報告してください

「広告ブロッカーを無効にしてください」

  1. ページをリロードします
  2. ProまたはTrialがあることを確認します（ステルスモードが必要）
  3. 続く場合：「ここで一時停止」を使用して報告してください

AdOffが有効にならない

chrome://extensions/に移動し、有効になっていることを確認します。エラーがある場合は、削除して再インストールしてみてください。

🔒 17. プライバシーとセキュリティ

AdOffは個人データを収集しません。あなたが訪問するサイトを追跡しません。テレメトリーを送信しません。第三者とデータを共有しません。広告を挿入しません。

権限とその理由

権限用途
storage設定とカウンターをローカルに保存する
declarativeNetRequest広告ネットワークへのリクエストをブロックする
activeTabサイト一時停止のためにドメインを知る
scripting広告を非表示にするコンテンツスクリプトを注入する
webNavigation適切なタイミングでブロックを有効化する
<all_urls>すべてのウェブサイトで機能する

外部通信

AdOffはあなたのアクションによってのみ外部サーバーと通信します：サイト報告、提案、ライセンスの有効化、自動再検証（Proは7日ごと）。通信はCloudflare Workerを経由します。

🗑 19. アンインストール

  1. chrome://extensions/に移動します
  2. AdOffを見つけて「削除」をクリックします
  3. 確認します

ローカル設定は削除されます。ProライセンスはサーバーでHasで有効のままです。紹介コードは失われます。

アンインストール前に、オプション > 詳細設定 > エクスポートから設定をエクスポートしてください。再インストール後、ファイルをインポートしてすべてを復元できます（ライセンスはキーで再有効化が必要です）。
