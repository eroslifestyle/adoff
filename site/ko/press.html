주요 기능

핵심 기능

  • 107개 이상의 규칙을 활용한 네트워크 수준 광고 차단 (declarativeNetRequest)
  • 고급 CSS 코스메틱 필터링 — 삽입된 광고 요소를 숨깁니다
  • 스트리밍 플랫폼 전반의 동영상 광고 자동 스킵
  • 쿠키 배너 및 팝업 제거
  • 추적기 및 핑거프린트 차단
  • 사이트별 화이트리스트 (1시간, 재시작 시까지, 또는 영구적으로 일시 중지)
  • 스텔스 안티-탐지 모드 — 광고 차단기 탐지 시스템에 보이지 않습니다 (Pro)
  • 베이트 스푸핑 및 변수 스푸핑 (Pro)
  • Fetch / XHR 인터셉션 (Pro)
  • 완전한 데이터 무수집 — 외부 서버로 아무것도 전송하지 않습니다
  • 추천 프로그램 — 무료 Pro 기간 획득
  • 6개 언어 UI (자동 감지 지원)

차별화 포인트

AdOff가 다른 이유

👻
진정한 스텔스 모드
대부분의 광고 차단기는 안티-애드블록 벽에 탐지됩니다. AdOff의 스텔스 모드는 브라우저의 MAIN world에서 실행되어, 탐지 스크립트가 아무 이상도 감지하지 못하도록 환경을 스푸핑합니다.
⚡
극도로 가벼운 용량
AdOff는 초경량으로 설계되어 레거시 부담이 없으며 Manifest V3를 위해 기본적으로 최적화되었습니다. 체감 로딩 시간을 전혀 추가하지 않으며 런타임 의존성이 없습니다.
🔒
완전한 데이터 무수집
AdOff는 브라우징을 추적하거나 데이터를 판매하거나 외부로 데이터를 전송하지 않습니다 — 선택적인 라이선스 확인 요청 한 건을 제외하고. 설계 단계부터 GDPR을 준수합니다.
💰
공정한 가격 모델
실질적으로 유용한 무료 티어를 제공하며, 유료 Pro 플랜은 월 2.99 EUR부터 시작합니다. 허용 광고 프로그램이나 스폰서 결과가 없습니다. 수익은 오직 사용자에게서만 나옵니다.
📺
동영상 광고 스킵 기능
MutationObserver를 사용하여 스킵 버튼이 활성화되는 순간 동영상 광고를 자동으로 스킵합니다 — 스트리밍 플랫폼에서 네트워크 수준 간섭이 필요하지 않습니다.
🌍
처음부터 다국어 지원
브라우저 언어 자동 감지를 통해 6개 언어로 전체 인터페이스와 지원 문서를 제공합니다. 유럽 우선 청중을 위해 설계되었습니다.

AdOff 소개

보일러플레이트 문구

AdOff에 대해 작성할 때 이 간단한 설명을 사용하세요.

AdOff는 Chromium 기반 브라우저를 위한 개인정보 우선 광고 차단기로, 모든 웹사이트에서 광고, 추적기, 동영상 광고를 차단하도록 설계되었습니다. 독자적인 스텔스 모드는 안티-애드블록 탐지 시스템에서 보이지 않게 만들어, 사용자 개입 없이 현대적인 광고 탐지 벽을 우회할 수 있는 몇 안 되는 브라우저 확장 프로그램 중 하나입니다. 초경량 설계로 Manifest V3에 기본적으로 최적화되어 있으며 사용자 데이터를 전혀 수집하지 않습니다. 30일 전체 Pro 체험판을 포함하여 무료로 제공되며, 월 2.99 EUR 또는 99 EUR의 평생 1회 구매로 유료 Pro 플랜을 이용할 수 있습니다. AdOff는 6개 언어를 지원하며 adoff.app에서 이용 가능합니다.
