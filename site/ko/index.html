AdOff의 력

여섯 가지 기술. 하나의 목표:
깨끗한 웹.

🚫
네트워크 수준의 완벽한 차단
→ 더 빠른 페이지. 더 적은 데이터 소비. 더 적은 추적기가 귀하를 프로파일링합니다.

138 네트워크 규칙이 광고 요청과 추적기를 브라우저에 도달하기 전에 제거합니다. 배너, 팝업, 리디렉션, 지문 수집: 로드되지 않습니다. 귀하는 관심 있는 콘텐츠만 봅니다.

🎬
비디오 광고 중립화 Pro
→ 즉시 시작하는 비디오. 카운트다운 없음, 중간 광고 없음.

AdOff는 비디오 플랫폼의 광고 SDK를 중립 버전으로 대체합니다. 플레이어는 즉시 "광고 없음" 신호를 받습니다.

👻
스텔스 안티-탐지 Pro
→ "광고 차단기 비활성화"를 다시 본 적 없음. 어떤 사이트에서도.

AdOff는 브라우저의 MAIN world에서 작동하여 모든 흔적을 숨깁니다: JavaScript 변수, 테스트 fetch, 미끼 요소. 안티-애드블록 시스템이 탐지하지 못합니다. 아무도 귀하를 막지 않고 계속 읽고 비디오를 볼 수 있습니다.

⚡
초경량 아키텍처
→ RAM과 배터리에 부담 제로. 단 몇 KB — 수 MB 차단기의 극히 일부입니다.

다른 차단기가 3-8MB이고 Chrome 메모리를 낭비하는 동안, AdOff는 그 일부입니다. 비동기 필터, 메인 스레드 차단 없음. 실행되는 것을 느끼지 못합니다. 브라우저가 다시 빨라진 것만 느낍니다.

🍪
쿠키 배너와 오버레이가 사라집니다
→ 사이트마다 '쿠키 허용'은 이제 그만. 첫 로딩부터 깨끗한 화면.

코스메틱 필터가 동의 배너, 팝업, 오버레이, 광고 섹션을 DOM에서 제거합니다. 페이지가 로딩되는 동안 사라지며, 닫을 필요가 없습니다. 읽기는 다시 당신의 것이 됩니다.

🔒
완전한 프라이버시, 제로 로그
→ 텔레메트리 없음, 프로파일링 없음. 당신이 무엇을 하는지 우리도 모릅니다.

브라우징 데이터는 기기를 떠나지 않습니다. 카운터, 화이트리스트, 환경설정은 로컬에 남습니다. 서버와의 유일한 연결은 Pro 라이선스 검증뿐입니다. 의무가 아닌 설계에 의한 GDPR 준수.

AdOff는 2026년 Chrome용 최고의 광고 차단 확장 프로그램 중 하나입니다. 네트워크 수준 차단(138개의 규칙), CSS 외형 필터링, 안티 애드블록 장벽을 우회하는 스텔스 탐지 회피 기술을 결합합니다. 배너, 팝업, 추적기, 동영상 광고를 차단하며, 대안보다 훨씬 가볍고 무료입니다, 신용카드 없이 30일 Pro 체험 포함.
네, 정말입니다. 무료 요금제는 영구적으로 무료이며, 시간 제한이 없고 신용카드가 필요하지 않습니다. 30일 Pro도 무료입니다: 결제 데이터를 요청하지 않습니다. 만료되면, Pro로 전환하지 않으면 자동으로 무료 요금제로 돌아갑니다. 숨은 비용 없음, 원치 않는 자동 갱신 없음.
예, IMA 광고 표준을 사용하는 모든 스트리밍 플랫폼에서. AdOff Pro는 광고 SDK를 중립 스텁으로 교체합니다: 플레이어는 '광고 없음'을 수신하고 비디오가 시작됩니다. 카운트다운 없음, mid-roll 없음. 비디오 광고 차단에는 AdOff Pro가 필요합니다, 30일 무료 평가판에 포함.
스텔스 모드 (체험에 포함되고 Pro 요금제에 포함)를 사용하면, AdOff는 브라우저의 MAIN world에서 작동하고 모든 흔적을 숨깁니다. 차단기 탐지에 사용되는 JavaScript 변수를 스푸핑하고, 광고 서버로의 테스트 fetch를 가로채고, 사이트가 예상하는 미끼 요소를 시뮬레이션합니다. 결과: 안티-애드블록 시스템이 활성 확장 프로그램의 신호를 받지 않습니다. 무료 요금제에는 스텔스가 포함되지 않으므로 가장 공격적인 사이트는 여전히 탐지할 수 있습니다.
네. 제로 로그 정책: AdOff는 귀하의 브라우징에 대한 데이터를 수집하지 않습니다. 저희는 귀하가 어떤 사이트를 방문하는지, 무엇을 검색하는지, 페이지에 얼마나 오래 머무르는지 모릅니다. 카운터, 허용 목록 및 선호도는 귀하의 기기에, chrome.storage.local 내에 유지됩니다. 저희 서버와의 유일한 통신은 Pro 라이선스 검증입니다 (키 + 이메일). 코드는 설치 전에 검사할 수 있습니다. 저희는 의무가 아닌 설계로 GDPR을 준수합니다.
Founder Lifetime은 한정 출시 혜택입니다. 한 번 99 €로 향후 모든 업데이트를 포함해 모든 개인 기기에서 AdOff Pro를 평생 사용합니다. 연간 갱신 없음, 깜짝 청구 없음, Founder 배지 포함. Founder 자리가 모두 차면 혜택은 종료됩니다.
네. AdOff는 Chrome, Firefox, Edge, Opera, Brave에서 사용할 수 있습니다(Safari 곧 지원). Pro 라이선스는 모든 개인 기기에서 유효합니다: 설치할 때마다 같은 라이선스 키를 입력하면 됩니다. 개인정보 보호를 위해 카운터는 각 브라우저에 로컬로 저장됩니다.
아니오, 반대입니다: 더 빠르게 만듭니다. AdOff는 초경량입니다 (경쟁사의 일부). 비동기 필터, 메인 스레드 차단 없음. 귀하는 그것을 실행하는 것을 느끼지 않습니다.
네. AdOff는 공식 채널 (Chrome 웹 스토어, Firefox Add-ons, Safari App Store)에 게시됩니다. 코드는 설치 전에 검사할 수 있습니다. GDPR 준수. 제로 로그 정책.
네. Free: 네트워크 차단 및 미용 필터는 모든 사이트에서 작동합니다. Pro: 비디오 광고 중립화는 표준 SDK를 사용하는 플랫폼에서 작동하며, 스텔스 모드는 안티-애드블록이 있는 사이트에서 필요합니다.
귀하가 망설이는 매 분마다, 수백 개의 추적기가 귀하에 대한 데이터를 수집합니다.
