🛠 1. AdOff란?

AdOff는 모든 웹사이트에서 광고를 차단하는 Google Chrome 확장 프로그램입니다. 네트워크, 페이지, 비디오 광고 무력화(Pro/Trial 전용), 스텔스(Pro/Trial 전용)의 4가지 수준으로 작동하여 깨끗하고 빠른 브라우징을 제공합니다.

  • 모든 사이트에서 광고 차단; 스트리밍 플랫폼의 비디오 광고 무력화 (Pro/Trial)
  • 스텔스 모드 (Pro/Trial): 안티-애드블록 사이트에서 AdOff를 보이지 않게 함
  • 초경량 — 네이티브 Manifest V3 설계
  • 개인 데이터 수집 없음: 모든 작동이 로컬에서 이루어짐
  • 6개 언어: 이탈리아어, 영어, 독일어, 프랑스어, 스페인어, 포르투갈어
  • 4가지 기간으로 사이트 일시정지: 방문, 1시간, 1일, 영구
  • 무료 Pro 일수를 얻는 추천 시스템

📥 2. 설치

Chrome 웹 스토어에서

  1. Chrome 웹 스토어에서 AdOff 페이지로 이동
  2. "Chrome에 추가" 클릭
  3. "확장 프로그램 추가"로 확인
  4. AdOff가 즉시 활성화됨

툴바에 아이콘 고정하기

  1. 브라우저 오른쪽 상단의 퍼즐 (확장 프로그램) 아이콘 클릭
  2. 목록에서 AdOff 찾기
  3. AdOff 옆의 핀 클릭
  4. 빠른 접근을 위해 툴바에 아이콘이 나타남

🎉 3. 첫 실행

처음 AdOff를 설치할 때:

  • 아이콘 고정 지침이 포함된 환영 페이지가 열림
  • 15일 Pro 트라이얼이 자동으로 활성화됨
  • 고유 추천 코드가 생성됨 (형식: ADO-XXXXX)
  • AdOff가 즉시 광고 차단 시작
  • 2026년 7월 1일 이전에 설치하면 영구 Founding Member 배지를 받음

🖼 4. 팝업

툴바의 AdOff 아이콘을 클릭하여 팝업을 엽니다.

헤더

로고, "AdOff" 이름, 라이선스 배지 (FREE / TRIAL Xdays / PRO) 및 FOUNDER 배지 (해당되는 경우).

전체 토글

모든 사이트에서 AdOff를 활성화 또는 비활성화하는 ON/OFF 스위치. 보라색 배지 = 활성, 빨간색 = 비활성화.

카운터

  • 차단된 광고: 페이지에서 제거된 광고 요소
  • 차단된 요청: 광고 서버로의 HTTP 요청 차단됨

숫자는 줄여서 표시됨: 1,500은 "1.5K", 2,300,000은 "2.3M".

현재 사이트

방문 중인 사이트의 도메인과 "여기서 일시정지" 버튼.

라이선스 배너

Free 사용자용("Pro 15일 무료 체험") 및 Trial 사용자용("트라이얼 활성화 — X일 남음")만 표시됨. Pro 사용자에게는 숨겨짐.

리뷰 프롬프트

100개 이상 광고 차단 후, 설치 후 5일 이상 경과 시 나타남. 최대 3번, 각 14일 간격. 세 가지 옵션: "리뷰 남기기", "나중에", "다시 보지 않기".

변경 로그

업데이트 후 새로운 기능을 표시. X를 클릭하여 닫음.

하단 링크

도움말 (옵션에서 FAQ 열기) 및 옵션 (전체 페이지 열기).

⏸ 5. 사이트 일시정지

  2. 툴바에서 AdOff 아이콘 클릭

  4. 기간 선택

참고: 만료된 일시정지는 자동으로 제거됩니다. 차단은 서브도메인에도 적용됩니다: example.com을 일시정지하면 blog.example.com도 제외됩니다.

⚠ 13. 문제 신고

AdOff와 함께 사이트가 작동하지 않으면 옵션 > 도움말 > 문제 신고에서 신고하세요.

  1. 사이트 URL 입력 (필수)
  2. 유형 선택: 사이트 작동 안 함 / 광고 보임 / 안티-애드블록 / 기타
  3. 설명 및 이메일 추가 (선택사항)
  4. 수학 캡차 풀기
  5. "신고 제출" 클릭

양식에는 안티-봇 보호 포함: 허니팟, 타이밍, 캡차, 속도 제한 (최대 3/시간), 안티-스팸, 중복 제거.

⚙ 15. AdOff 작동 원리 (기술적)

레벨 1 — 네트워크 차단

AdOff는 Chrome의 declarativeNetRequest API를 사용하여 브라우저가 다운로드하기 전에 광고 서버로의 HTTP 요청을 차단합니다. 120개 이상의 규칙이 주요 광고 네트워크와 많은 다른 광고 네트워크를 포함합니다.

레벨 2 — 코스메틱 필터링

사이트와 동일한 도메인에서 제공되는 광고의 경우, AdOff는 CSS와 JavaScript를 사용하여 페이지에서 숨깁니다: 정밀 선택자, 동적 콘텐츠를 위한 MutationObserver, 빈 컨테이너 축소.

레벨 3 — 비디오 광고 무력화 (Pro/Trial 전용) PRO

대부분의 비디오 플레이어는 광고 중단(프리롤, 미드롤, 포스트롤)을 위한 표준 광고 SDK를 사용합니다. AdOff는 이를 플레이어에게 "광고 없음"을 즉시 전달하는 중립 버전으로 교체합니다 — 중단 없이 비디오가 시작됩니다. 전 세계 모든 사이트에서 자동으로 작동합니다. 주요 비디오 플랫폼에서: 16배 가속 + 자동 "광고 건너뛰기" 클릭. Pro 전용 기능.

레벨 4 — 스텔스 모드 (Pro/Trial 전용)

안티-애드블록 확인을 가로채어 사이트가 모든 것이 보인다고 믿게 합니다:

  • CSS 스푸핑: 미끼 요소 속성 위조
  • 네트워크 인터셉션: 감지 스크립트가 빈 응답을 받음
  • 변수 시뮬레이션: 광고 SDK 사전 채우기
  • 스크립트 무력화: 안티-애드블록 스크립트가 실행 전에 차단됨
  • 안티 스크롤 잠금: 벽이 스크롤을 차단하는 것을 방지

스텔스는 주요 플랫폼 (비디오, 소셜, 이커머스) 및 간섭할 수 있는 다른 대형 사이트에서 자동으로 비활성화됩니다.

🛠 16. 문제 해결

사이트가 작동하지 않음

  1. AdOff 아이콘을 클릭하고 "여기서 일시정지" 사용
  2. 페이지 새로고침
  3. 일시정지 중에 작동하면: 사이트를 신고해주세요
  4. 일시정지 중에도 작동하지 않으면: 문제가 AdOff와 관련이 없음

여전히 광고가 보임

  1. AdOff가 활성화되어 있는지 확인 (보라색 배지)
  2. 사이트가 화이트리스트에 없는지 확인
  3. 페이지 새로고침
  4. 지속되면: 사이트를 신고해주세요

"광고 차단기를 비활성화해주세요"

  1. 페이지 새로고침
  2. Pro 또는 Trial이 있는지 확인 (스텔스 모드 필요)
  3. 계속되면: "여기서 일시정지" 사용하고 신고해주세요

AdOff가 활성화되지 않음

chrome://extensions/로 이동하여 활성화되어 있는지 확인. 오류가 있으면 제거 후 재설치해보세요.

🔒 17. 개인정보 및 보안

AdOff는 개인 데이터를 수집하지 않고, 방문하는 사이트를 추적하지 않으며, 원격 측정을 보내지 않고, 제3자와 데이터를 공유하지 않으며, 광고를 주입하지 않습니다.

권한 및 이유

권한용도
storage설정 및 카운터를 로컬에 저장
declarativeNetRequest광고 네트워크 요청 차단
activeTab사이트 일시정지를 위한 도메인 확인
scripting광고 숨기기를 위한 콘텐츠 스크립트 주입
webNavigation적시에 차단 활성화
<all_urls>모든 웹사이트에서 작동

외부 통신

AdOff는 사용자의 조치에 의해서만 외부 서버와 통신합니다: 사이트 신고, 제안, 라이선스 활성화, 자동 재검증 (Pro의 경우 7일마다). 통신은 Cloudflare Worker를 통해 이루어집니다.
