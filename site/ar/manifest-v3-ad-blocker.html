حاجب إعلانات Manifest V3 اللي لسه يشتغل

Manifest V3 غيّر كيف Chrome تحجب الطلبات، ضاعفت حاجبات مبنية على نظام قديم. حاجب Manifest V3 أصلي زي AdOff مبني على محرك القوانين الجديد من الأول، حتى يبقى يحجب الإعلانات كل الكفاية والكفاية.

شنو Manifest V3 غيّر

Manifest V3 بـ Chrome بدّل API قديم لحجب الطلبات الديناميكي بـ واحد معاصر وأغلق كيفية القوانين. حاجبات مبنية على النهج القديم اضطرت إن تعاد كتابتها، وبعض خسرت قوة أو انسحبت، وهذا ليش مستخدمين طويلين لاحظوا الإعلانات ترجع.

ليش حاجبات MV3 أصلية تبقى قوية

حاجب مبني على Manifest V3 يستخدم محرك القوانين المعاصر اللي Chrome بدته الطريقة اللي Chrome بدها الآن، مع فلترة تجميلية content-script وسكريبت main-world لحالات متقدمة. AdOff مبني MV3 أصلي: قوانين شبكة، إخفاء تجميلي، تحييد إعلانات فيديو و Stealth كل واحد يشتغل داخل النموذج الجديد.

شنو الفحص قبل التركيب

تأكد إن الحاجب يدعم Manifest V3 بشكل صريح وفي صيانة فعالة. AdOff MV3 أصلي عبر Chrome وEdge وOpera وBrave زائد نسخ Firefox وSafari.
