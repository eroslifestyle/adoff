🧭 Before you start Safari is stricter than other browsers. It requires Xcode (Apple's free app) to install extensions. The procedure takes about 30 minutes the first time. Requires macOS Ventura 13+, Safari 16.4+. Don't worry if you've never opened Terminal, I'll guide you step by step.
1
Extract the ZIP file
1.1 Apri Finder (l'icona sorridente nel Dock in basso).
1.2 Nella barra sinistra, clicca Download.
1.3 Trova il file adoff-safari.zip (quello che hai appena scaricato).
1.4 Doppio click sul file. macOS lo estrarrà automaticamente e creerà una cartella adoff-safari accanto allo ZIP.
1.5 Non spostare o rinominare questa cartella. Lascila in Download.
2
Install Xcode (free, ~10 GB)
2.1 Apri Mac App Store (icona blu con "A" bianca, nel Dock o Launchpad).
2.2 Nella barra di ricerca in alto a sinistra, scrivi Xcode e premi Return.
2.3 Clicca Scarica o Installa sulla voce Xcode (icona Apple, di Apple).
2.4 Accedi con il tuo Apple ID se richiesto (lo stesso dell'App Store).
2.5 Attendi il download. Xcode è grande (~10 GB), potrebbe richiedere 20-60 minuti.
2.6 Quando è pronto, apri Xcode almeno una volta: clicca l'icona in Launchpad o Applicazioni.
2.7 Appare una finestra con i termini di licenza. Clicca Accetta.
2.8 Attendi che Xcode finisca la configurazione iniziale (1-2 minuti), poi chiudilo (Cmd+Q).
💡 If your Mac doesn't have enough space, free at least 15 GB before starting.
3
Open Terminal and run command
3.1 Premi Cmd + Space insieme. Si apre una barra di ricerca (Spotlight) in alto.
3.2 Scrivi Terminal e premi Return. Si apre una finestra bianca/nera (Terminal, niente di spaventoso, è un programma Mac normale).
3.3 Copia questo comando esattamente (clicca il box grigio per selezionarlo, poi Cmd+C):
xcrun safari-web-extension-converter ~/Downloads/adoff-safari
3.4 Clicca nella finestra Terminal, incolla con Cmd+V, poi premi Return.
3.5 Terminal chiede "Continue?". Scrivi y e premi Return.
3.6 Attendi 30-60 secondi. Xcode si apre automaticamente con un progetto "AdOff".
3.7 In Xcode, clicca il pulsante ▶ Play in alto a sinistra (oppure Cmd+R).
3.8 Compilazione: attendi 1-3 minuti. Potrebbe aprirsi una piccola finestra "AdOff". Puoi chiuderla (l'estensione è già installata).
⚠ If you get "permission denied" or "command not found": open Xcode, Preferences, Locations, Command Line Tools, and select the newest version. Then retry step 3.4.
4
Enable Developer menu in Safari
4.1 Apri Safari (icona bussola nel Dock).
4.2 Nella barra menu in alto, clicca Safari → Impostazioni (oppure Cmd+,).
4.3 Si apre una finestra con più tab in alto. Clicca su Avanzate (l'ultimo, icona ingranaggio).
4.4 In basso, trova l'opzione Mostra funzioni per sviluppatori web (o "Mostra menu Sviluppo" nelle versioni più vecchie).
4.5 Seleziona la casella. Appare un nuovo menu Sviluppo nella barra menu di Safari.
4.6 Chiudi la finestra Impostazioni.
5
Allow unsigned extensions
5.1 Nella barra menu di Safari, clicca sul nuovo menu Sviluppo.
5.2 Scorri il menu e clicca Consenti estensioni non firmate.
5.3 Safari chiede la tua password Mac (quella che usi per accedere). Scrivila e premi Return.
5.4 L'opzione è ora attiva (puoi verificare: riaprendo il menu Sviluppo, "Consenti estensioni non firmate" ha un segno di spunta).
⚠ Important: this option disables automatically every time you restart Safari. You must re-enable it (step 5.1-5.3) every system restart.
6
Enable AdOff
6.1 Riapri le Impostazioni di Safari: Safari → Impostazioni (oppure Cmd+,).
6.2 Clicca sulla tab Estensioni (icona puzzle).
6.3 Nell'elenco a sinistra, dovresti vedere AdOff. Seleziona la casella accanto.
6.4 Appare un popup che chiede i permessi di accesso ai siti. Clicca Consenti sempre su tutti i siti (o "Consenti su tutti i siti").
6.5 Chiudi le Impostazioni. Visita qualunque sito con ads (news, piattaforma video, forum). Gli ads sono bloccati.
6.6 L'icona AdOff (una "A" con overlay rosso) appare automaticamente nella barra di Safari in alto a destra. Clicca per aprire il popup con i controlli.
✅ Done. AdOff is active. You get 30 free Pro trial days. The icon shows the countdown.

Preguntas frecuentes

Do I have to keep the folder on my computer? ▼
Yes. The browser reads the extension files from the folder. If you delete it, the extension stops working. Put it somewhere safe (e.g., Documents).
Is it safe to enable Developer Mode? ▼
Yes, completely safe. Developer Mode just lets you install extensions from a folder instead of the store. It doesn't change anything else in your browser.
The browser says "Disable developer extensions" - what do I do? ▼
Some browsers show a warning on startup when you have manually installed extensions. Just close the warning. The extension keeps working normally. When AdOff gets to the official store, this warning disappears.
How do I update AdOff? ▼
Download the new version from the site, extract the ZIP in the same folder (overwrite the files), then go to the extensions page in your browser and click the refresh arrow on AdOff.
Does it work on all websites? ▼
Yes. AdOff blocks ads on all websites: video platforms, social media, news, blogs, any site. You can also pause blocking on specific sites if needed.
How do I uninstall AdOff? ▼
Go to your browser's extensions page, find AdOff, and click "Remove". Then you can delete the folder from your computer.
Which browsers are supported? ▼
AdOff supports all Chromium browsers (Chrome, Edge, Brave, Opera, Vivaldi, Arc) with 100% feature compatibility, Firefox with 95%, and Safari 16.4+ on macOS Ventura+ with 95% (requires Xcode). On Android, use Kiwi Browser or Firefox. iOS support coming soon.
Does it work on mobile? ▼
On Android, install Kiwi Browser (free from Play Store) which supports Chrome extensions. On iOS, browser extensions are not yet supported. We're working on it.
