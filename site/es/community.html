¿Encontraste un anuncio que se cuela? ¿Un sitio que se rompe? Abre un reporte de error en GitHub o Discord y lo investigaremos. Incluye la URL del sitio y los pasos para reproducirlo.

Sugerir una Función
¿Tienes una idea que mejoraría AdOff? Compártela en nuestras discusiones de GitHub o en Discord. Las mejores ideas llegan a la próxima versión.

Una reseña en la Chrome Web Store ayuda a más usuarios a descubrir AdOff. Tarda 30 segundos y marca una enorme diferencia para un proyecto independiente como el nuestro.
