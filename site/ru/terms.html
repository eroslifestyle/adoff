1. Принятие условий

Устанавливая, получая доступ или используя расширение AdOff для Chrome («Расширение»), сайт adoff.app («Сайт») или API api.adoff.app («API») (совместно — «Сервис»), вы соглашаетесь соблюдать настоящие Условия использования («Условия»). Если вы не согласны с Условиями, не используйте Сервис.

Настоящие Условия представляют собой юридически обязательное соглашение между вами и AdOff. Мы оставляем за собой право обновлять Условия в любое время (см. §15 о требованиях к уведомлению).

2. Описание Сервиса

AdOff — расширение для браузера Chrome, которое блокирует рекламу на уровне сети и визуальном уровне и предотвращает обнаружение расширения системами защиты от блокировщиков. Сервис включает:

  • Расширение AdOff для Chrome — программное обеспечение, установленное в вашем браузере
  • Сайт (adoff.app) — информационные страницы и страницы покупки
  • API (api.adoff.app) — проверка лицензии и обработка покупок

3.1 Бесплатный план

Бесплатный план доступен всем без регистрации, оплаты или создания учётной записи. Включает базовую функциональность блокировки рекламы, описанную выше.

3.2 Pro план

Plan Pro требует приобретения лицензионного ключа. Функции Pro активируются вводом лицензионного ключа в расширении. Лицензия Pro позволяет одновременную активацию на 3 устройствах. При попытке активации более чем на 3 устройствах потребуется предварительно деактивировать одно из существующих.

3.3 Система аккаунта (необязательно)

Пользователи могут создать аккаунт AdOff для управления устройствами и лицензиями через единую панель на adoff.app/account. Создание аккаунта не обязательно ни для одного тарифного плана, включая Pro.

Доступные способы входа:

  • Вход через Google — войдите с помощью аккаунта Google
  • E-mail + пароль — зарегистрируйтесь с адресом e-mail и паролем
  • Лицензионный ключ + e-mail — войдите с помощью ключа Pro-лицензии и e-mail покупки (пароль не нужен)

Данные аккаунта регулируются нашей Политикой конфиденциальности. Бесплатная версия не требует аккаунта и не предполагает связи с сервером.

3.4 Управление устройствами

Каждая установка браузера считается одним устройством. Пользователи Pro могут просматривать активные устройства и управлять ими на adoff.app/account. Для соблюдения лимита в 3 устройства пользователи могут удалять устройства из панели управления. Действует ограничение не более 3 удалений устройств в день для предотвращения злоупотреблений.

3.5 Бесплатный пробный период

Новые установки включают 15-дневный бесплатный пробный период всех функций Pro. Кредитная карта не требуется. Пробный период начинается автоматически при первой установке и не может быть перезапущен после истечения.

4. Покупка лицензии и цены

Лицензии Pro предлагаются в виде ежемесячных подписок, годовых подписок или пожизненных лицензий. Актуальные цены отображаются на сайте adoff.app на момент покупки.

  • Цены на подписки могут изменяться. Изменения будут сообщены не менее чем за 30 дней и не затронут текущий расчётный период.
  • Существующие подписки соблюдаются по цене, действовавшей на момент покупки, для текущего расчётного периода. Продления после изменения цены будут выставляться по новой ставке после обязательного периода уведомления.
  • Пожизненные лицензии — единовременная покупка. Они действительны на весь срок существования продукта AdOff для первоначального покупателя и не подлежат передаче третьим лицам.

30-дневная гарантия возврата денег: Если вы по какой-либо причине не удовлетворены лицензией Pro, свяжитесь с нами через страницу поддержки в течение 30 дней с момента покупки для получения полного возврата. Без вопросов.

6. Совместимость и производительность

AdOff разработан для блокировки рекламы на широком спектре сайтов. Однако:

  • Мы не гарантируем, что вся реклама на всех сайтах будет заблокирована. Рекламные сети постоянно развивают механизмы доставки.
  • Совместимость с конкретными сайтами, включая видеостриминговые платформы и новостные ресурсы, поддерживается по мере возможности и может быть затронута обновлениями этих сервисов.
  • AdOff работает как расширение Chrome и подвержен ограничениям браузера, включая ограничения Manifest V3.
  • Для поддержания эффективности по мере изменения рекламных сетей могут потребоваться обновления расширения. Обновления доставляются через Chrome Web Store.
