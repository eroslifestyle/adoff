Политика конфиденциальности

Последнее обновление: 21 апреля 2026 г.  ·  Версия 2.0

Кратко: AdOff не собирает личные данные о просмотре. Не отслеживает вас. Не продаёт данные никому. Единственная коммуникация с нашими серверами — проверка лицензии Pro, и даже там мы передаём минимально необходимое. Версия FREE работает без регистрации и учётной записи — данные бесплатных пользователей не собираются, кроме данных, хранящихся локально на устройстве.

1. Кто мы

AdOff — расширение для браузера, блокирующее рекламу и трекеры. Ответственным за обработку данных является команда AdOff, с которой можно связаться через страницу поддержки.

2. Данные, которые мы не собираем

AdOff не собирает, не передаёт и не хранит:

  • Историю посещённых сайтов или URL посещаемых страниц
  • Содержимое страниц или данные, введённые в формы
  • Сторонние cookie для профилирования или отслеживания
  • Поведенческие данные или данные профилирования
  • Любые данные, которые могут быть использованы для вашего профилирования без вашего согласия

Версия FREE работает без регистрации и учётной записи. Данные бесплатных пользователей не собираются, кроме данных, хранящихся локально на устройстве.

3. Данные учётной записи (только для пользователей Pro)

Если вы создаёте учётную запись или приобретаете план Pro, мы собираем следующие данные:

  • Адрес электронной почты — собирается при покупке (через Stripe) и при регистрации учётной записи. Используется для: доставки лицензии, входа в учётную запись, служебных сообщений. Правовое основание: исполнение договора (ст. 6(1)(b) GDPR).
  • Пароль — хранится в виде необратимого хэша (SHA-256 + соль). Никогда в открытом виде. Никогда не передаётся третьим лицам.
  • Google OAuth — если вы используете «Войти через Google», мы получаем ваш адрес электронной почты Google и идентификатор профиля. Мы не получаем доступ к другим данным вашего аккаунта Google.
  • UUID устройства — случайный идентификатор, генерируемый при установке расширения. Используется для применения ограничений по количеству устройств на лицензию. Не связан с вашей личностью. Правовое основание: законный интерес (ст. 6(1)(f) GDPR).

4. Данные, хранимые локально

AdOff использует chrome.storage.local и chrome.storage.sync для локального хранения на вашем устройстве:

  • Лицензионный ключ Pro (при наличии покупки)
  • Настройки расширения (например, белый список сайтов, параметры интерфейса)
  • Состояние активации (активно/отключено для сайта)

Эти данные никогда не отправляются на сторонние серверы и остаются в вашем браузере. chrome.storage.sync может синхронизировать данные между устройствами через ваш аккаунт Google согласно политикам Google.

5. Проверка лицензии

Если вы приобрели план Pro, расширение обращается к нашим серверам для проверки действительности лицензии. При этом передаются:

  • Лицензионный ключ
  • UUID устройства
  • Идентификатор браузера/ОС (для панели управления устройствами)
  • Версия расширения

Ваш IP-адрес виден нашему серверу (Cloudflare) но не хранится постоянно — из него извлекается только геолокация на уровне страны/города для агрегированной аналитики. Журналы доступа к серверу удаляются в течение 30 дней.

6. Третьи лица

AdOff не передаёт, не продаёт и не уступает данные третьим лицам в маркетинговых или профилировочных целях. Мы не используем:

  • Сервисы поведенческой аналитики
  • Рекламные сервисы
  • Сторонние SDK отслеживания

Сторонние сервисы, которые мы используем для предоставления услуги:

  • Stripe — обработка платежей. Платёжные данные обрабатываются Stripe в соответствии с их собственной Политикой конфиденциальности.
  • Google Sign-In — если вы используете «Войти через Google», ваша аутентификация обрабатывается Google. Ознакомьтесь с политикой конфиденциальности Google.
  • Cloudflare — хостинг и доставка контента. Cloudflare может обрабатывать IP-адреса в соответствии со своей политикой конфиденциальности.

7. Разрешения расширения

AdOff запрашивает следующие разрешения браузера, все необходимые для работы:

  • declarativeNetRequest — для блокировки сетевых запросов (реклама, трекеры)
  • storage — для локального сохранения настроек и лицензии
  • tabs — для применения настроек к активной вкладке
  • scripting — для внедрения CSS скрытия на веб-страницах
  • webNavigation — для повторного применения фильтров при переходе между страницами

Ни одно из этих разрешений не используется для сбора или передачи данных о просмотре.

8. Ваши права (GDPR)

В соответствии с Общим регламентом защиты данных ЕС вы имеете право на:

  • Доступ — запросить копию данных, которые мы храним о вас
  • Исправление — запросить исправление неточных данных
  • Удаление — запросить удаление ваших данных («право на забвение»)
  • Переносимость данных — получить ваши данные в переносимом формате
  • Ограничение — запросить ограничение обработки
  • Возражение — возразить против обработки ваших данных
  • Жалоба — подать жалобу в надзорный орган в Европейском Союзе
  • Удаление учётной записи — удалить вашу учётную запись и все связанные данные
  • Экспорт данных — экспортировать данные вашей учётной записи
  • Управление устройствами — управлять лицензированными устройствами в adoff.app/account

Для реализации ваших прав свяжитесь с нами через страницу поддержки.

9. Хранение данных

  • Локальные настройки: хранятся на вашем устройстве до удаления расширения или очистки данных браузера.
  • Данные учётной записи (электронная почта, хэшированный пароль): хранятся пока учётная запись активна. Удаляются по запросу.
  • UUID устройства: удаляется при удалении устройства из панели учётной записи.
  • Геолокация, полученная из IP: агрегируется только на уровне страны. Отдельные IP-адреса не хранятся постоянно.
  • Хэшированные пароли: удаляются при удалении учётной записи.
  • Журналы проверки лицензии: автоматически удаляются в течение 30 дней.
  • Платёжные данные: управляются нашим платёжным процессором согласно его политике хранения.

10. Правовое основание обработки

Правовым основанием для обработки данных учётной записи и лицензии является исполнение договора (ст. 6(1)(b) GDPR) — это необходимо для проверки вашей лицензии Pro и предоставления приобретённой услуги. Обработка UUID устройства основана на законном интересе (ст. 6(1)(f) GDPR) — необходимо для применения ограничений по количеству устройств на лицензию.

11. Несовершеннолетние

AdOff не предназначен для лиц младше 13 лет. Мы сознательно не собираем данные от несовершеннолетних.

12. Изменения в политике

Мы можем обновлять эту Политику конфиденциальности в будущем. О существенных изменениях будет сообщено через расширение или сайт. Продолжая использовать AdOff после изменения, вы принимаете новую версию политики.

14. Добровольные Диагностические Данные

AdOff включает функцию опциональной отправки отчётов об утечке рекламы: когда расширение обнаруживает видимую рекламу и вы решаете отправить отчёт, на наши серверы отправляется диагностический пакет для анализа. Эта функция полностью опциональна и отключена по умолчанию. Диагностические данные никогда не отправляются без вашего явного согласия. При отправке отчёта передаются следующие данные:

  • URL страницы, на которой была обнаружена реклама
  • Версия установленного расширения AdOff
  • Технические данные о видимом рекламном элементе (CSS-селектор, позиция)
  • Список активных правил фильтрации (не история браузера)
  • Базовая информация о браузере и операционной системе
Эти данные используются исключительно для улучшения правил блокировки рекламы. Они не связаны с вашей личностью, не продаются и не передаются третьим лицам. Диагностические отчёты хранятся не более 90 дней, затем удаляются.
