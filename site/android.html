Install in 3 steps

Works on Android 10 and above

  1. Enable "Install unknown apps"

    Go to Settings → Apps → Special access → Install unknown apps. Find your browser (Chrome, Firefox, etc.) and enable it. This lets you install the AdOff APK.

  2. Download the APK

    Tap the download button below. The file adoff-android.apk will be saved to your Downloads folder.

  3. Open and install

    Tap the downloaded file in your Downloads app. Android will show a warning — tap Install anyway. AdOff will request VPN permission on first launch. Tap OK to allow.
